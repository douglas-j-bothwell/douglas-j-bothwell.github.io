Database Browsing Insight
The Database Browsing insight is useful when you want to find SQL statements, query commands, and other database information of interest.
Figure 1 Database Browsing Insight
 
With this insight, you can
Find top database servers and clients based on criteria such as response times, amount of data returned, and rate of kilobytes or packets returned.
Filter the transactions that are displayed by matching the SQL text against either a text string or a regular expression (to use a regular expression, prepend the search string with the keyword “regex:”).
Drill down into detailed data for specific databases, SQL statements, query commands, and other types of data.
Troubleshoot SQL statements of interest—Right-click on a SQL statement in the table and choose Troubleshoot SQL Statement. This opens the statement in the Database Statement Troubleshooting Insight.
View a time series chart for a specific metric—Double-click on a metric cell in the table, such as Average Response Time (Final Response) (sec).
Data in the main table is organized into these groups:
 
Table 1
Data Groups
Group
Description
SQL Statement

DB Server

DB Instance

Client User

Application
