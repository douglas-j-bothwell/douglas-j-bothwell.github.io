Conference Dashboard
The Conference Dashboard (Insights > Video > Conference Dashboard) provides a general overview of all conferences during the selected time window.
Figure 2 Conference Dashboard Insight
 
Note the following:
This dashboard includes some special metrics that are described in Table 1.
Each conference is a set of concurrent calls from two or more users to the same callee.
By default, this insight shows the top 100 conferences by lowest cumulative MOS score; the list is ordered initially by the amount of time in which calls were in a degraded—Fair, Bad, or Worst—state.
Quality thresholds are defined in the VoIP web UI > VoIP Data Settings > Call Quality Thresholds page.
You can set the Show menu to view all observed conferences or only those that are currently active.
You can right-click on a conference to
–
View Call Details—opens the VoIP Call Record Lookup Insight, with detailed information about each call in the selected conference
–
View Timeline of Call Quality—opens the Conference Quality Timeline Insight, with a graphical display of call quality for the conference
 
Table 1
Conference Dashboard Metrics
Metric
Description
Conference Key
The SIP address of the conference.
Call Quality of Last Observation
The call quality when the appliance last observed the call (the appliance samples call quality at one-minute intervals).
Time in Degraded State
The amount of time when the call was in a Fair, Bad, or Worst state.
Cumulative MOS
The overall MOS score for the call, from the start of the call to the end (or the current time, if the call is still active).
Worst MOS Observed
The worst MOS score observed for any one-minute interval during the call.
