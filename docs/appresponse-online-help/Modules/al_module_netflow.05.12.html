Viewing NetFlow Interface Data
The NetFlow Module adds the following features for viewing NetFlow Interface data. (These features are available only when NetFlow data collection is turned on, as described in Setting Up NetFlow Data Collection.)
Interface Table
This table lets you view the top interfaces based on interface-specific metrics such as inbound/outbound traffic, utilization, and throughput.
NetFlow Insights
These insights (accessed by choosing Insights > NetFlow) leverage NetFlow data to let you find specific interfaces of interest:
–
Bandwidth Hogs by Interface
–
High Link Utilization by Application
–
Interface Search by Application
–
Interface Search by IP
–
Riverbed NetFlow WAN Performance
–
Riverbed NetFlow WAN Troubleshooting
–
Riverbed NetFlow WAN Usage
–
Selected Interface Table
–
Top IPs by Interface
NetFlow QoS Insights
These insights use NetFlow data to let you find specific interfaces by application and ToS:
–
Applications for Selected ToS
–
Applications for Selected ToS Group
–
ToS Groups for Selected Application
–
ToS for Selected Application
Note: NetFlow Insights might not be pre-installed on your Appliance, in which case you will need to download them from the Update Center. To do so, go to the AppResponse Desktop Console, choose Insights > Update Center, and select the following URL in the Server pull-down menu:
support.opnet.com/ace_live/insights/netflow/
