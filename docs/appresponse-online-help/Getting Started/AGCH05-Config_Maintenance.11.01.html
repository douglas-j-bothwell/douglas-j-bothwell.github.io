Administration and Maintenance
With the AppResponse Appliance installed, you can now focus on more advanced configuration and maintenance (see Installing the Appliance). The configuration procedures covered in this chapter are used to manage advanced features of the appliance, but are not required for basic operation of the system. These procedures can be performed at any time after initial installation of the appliance.
