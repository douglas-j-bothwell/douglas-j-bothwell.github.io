Web Console Enhancements
This release includes the following enhancements:
The Web Console includes a new Search field (top right) that enables you to search configuration and traffic data on the appliance.
For more information, see Global Search in Web Console.
The Individual Page Views navigator includes a Waterfall graph that shows the Server Busy Time for individual resources and for the entire page. This view is useful when you want to identify when server processing delays contribute significantly slow page times. Previously, waterfall graphs were available only in the Desktop Console > Individual Page Views insight.
For more information, see Waterfall Graph in Individual Page Views Navigator.
