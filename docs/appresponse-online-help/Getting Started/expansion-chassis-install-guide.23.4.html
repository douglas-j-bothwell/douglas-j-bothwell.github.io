Viewing RAID Status on the Appliance
The following utilities provide information about the operation of attached disk arrays:
raid Utility
tw_cli Utility (4100, 4200, and 5000 Appliances Only)
raid Utility
raid is a command-line program that is useful for viewing high-level array and device information for both internal and external RAID controllers. To view high-level information about all RAID controllers, arrays, and devices, do the following:
1)
Open a CLI window and log in to the appliance as admin.
2)
Run the command:
raid
3)
From the RAID menu, run the command:
status

tw_cli Utility (4100, 4200, and 5000 Appliances Only)
tw_cli is a command-line utility for RAID controllers. This utility is not supported on 4300, 5100, or 6000 appliance models.
You can run this utility from the CLI using the raid command as described in raid Utility. To view the status for a specific controller, run the command:
tw_cli /cn show
where n is the controller number. For example, for the status of Controller 0, enter:
tw_cli /c0 show
