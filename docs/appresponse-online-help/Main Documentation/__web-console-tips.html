Tips and Tricks
Use the following tips and tricks to customize the Web Console to your specific workflows:
Create Favorites Views to get quick access to your key data.
Every Navigator table has a Send To > New Favorites View operation: select the groups you want and then send it to a new or current Favorites View. You can use Favorites Views to ensure that the data you are most interested in is immediately accessible.
 
To save the layout when you log out or close the window, set the View to Open preference to Open Workspace (Figure 2).
Rename your Favorites Views to something meaningful.
To rename a view, right-click in the tab or click in the space between the title and the Close (X) button.
 
You can close the sidebar to maximize the view space. This is especially useful when viewing on small laptops. You can also configure the Console to show the sidebar in the Home tab only (as noted in Home and Side Bar).
To access the web UI for system administration, choose Administration > System in the sidebar.
To edit Custom Views and SLA Dashboards quickly while viewing, choose Settings > Edit in the top-right corner.
 
To navigate around the Console quickly, use the breadcrumb at the top.
The Custom Views master page includes a Search field that you can use to find Business Groups and Applications easily.
