Selector Components
Selectors are used to pick user-defined groups, primitive groups (e.g., IP groups), and tasks from lists of user-defined groups and tasks.There are eight types available for building Insights:
Alert Selector
Autonomous Systems Selector
Business Application Selector
Business Group Selector
Combo Business Group Selector
Custom Link Selector
Drilldown Selector
Interface Group Selector
Interface Group Search Selector
IP Address Selector
User-Defined Group Selector
Selection Mode
In all Selector Components, you may specify the selection mode by choosing from the Selection drop-down menu in the Component edit dialog. There are three selection modes:
In Single mode, only one item can be selected.
In Contiguous mode, multiple items can be selected, but they must be contiguous or next to each other.
In Discontiguous mode, multiple items can be selected and they do not need to be next to each other.
The selection mode is specified when the Insight is created and cannot be changed by the user.
Business Application Selector
The Business Application Selector is used to select Defined Applications. It shows a list of Defined Applications.
Edit the Business Application Selector
You may configure the selection mode of the Business Application Selector, as described in Selection Mode, and you may select the items displayed upon startup, depending on the selection mode.

Alert Selector
The Alert Selector is used to select Static, Static-Any, and Adaptive Alerts. It displays a hierarchical list of the three types of alerts.
Edit the Alert Selector
You may configure the selection mode of the Alert Selector (as described in Selection Mode), and you may select the items displayed upon startup, depending on the selection mode.
Figure 66
Edit Alert Selector
 
The Alert Selector is intended for use in selecting an Alert task (i.e., the definition of an Alert) and sending it to a script. No other documentation is available at this time.
User-Defined Group Selector
The User-Defined Group Selector is used to select User-Defined Groups, which include Business Groups, Defined Applications, Business Group Links (on a appliance only), and Director Groups (on a Director only).
Edit the User-Defined Group Selector
You may configure both the type of group(s) displayed, the selection mode of the User-Defined Group Selector (as described in Selection Mode), and you may select the items displayed upon startup, depending on the selection mode.
Figure 67
Edit User-Defined Group Selector
 
The selector can display one of the User-Defined Group types by itself, or all of the User-Defined Group types together in a tabbed pane where each type of group has its own tab. This choice is made when the Insight is created and cannot be changed by the user.
