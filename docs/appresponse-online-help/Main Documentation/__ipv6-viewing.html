Viewing IPv6 Information
AppResponse can collect, analyze, and display both IPv4 and IPv6 traffic. Once IPv6 is enabled on the appliance, you can view IPv6 data in the tables, charts, and reports generated by AppResponse.
Generally, IPv6 information is added automatically. If you have an existing appliance to which you are adding IPv6 support for the first time, you might wish to update your Business Group definitions and Favorites to include IPv6.
