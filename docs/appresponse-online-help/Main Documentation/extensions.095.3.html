Running an Insight Extension
The following components can be used to run an Insight Extension:
Launcher Visual Component
–
Scripts launched by the user from an HTML document
- ascript - Insight Extension with output in the component
- iscript - Local Script with output in the component
- script - Local Script with output in a separate console window
–
Scripts launched directly by the launcher
- Local Script - Local Script with output in the component
- Remote Script - Insight Extension with output in the component
Script Analysis
–
Local Script - Local Script with no output
–
Remote Script - Insight Extension with no output
