Interactions
The following table lists each Interaction, sorted by its display name as it appears in the Insight Development Kit Component Interactions pane, along with a brief description of its purpose.
 
Table 3
Interactions 
Interaction
Description
alert
Passes alert definitions between Components.
analysisResult
Passes analysis results between Components.
box
Passes the name of the Appliance that is to be queried between Components.
clear
Notifies other Components that the Clear Control was clicked in the Toolbar Component.
group
Passes a set of traffic groups or applications between Components. The consuming Component can use the group index to pick a specific group from that set.
For more information, see Using the Group Index.
groupValues
Passes a traffic group and its traffic data between Components.
index
Passes a Component index between Components. Currently used for highlighting.
insight
Passes Insight references between Components.
interval
Passes a time interval between Components. A null interval is used to signify that the interval has reverted to the Bulletin Board interval.
metric
Passes a set of metrics between Components. An empty or null set of metrics indicates that there are no metrics.
singleQuery
Sends out start and end notifications. In between the start and end notification, no queries should be launched.
text
Passes a text string between Components.
timeData
Passes external time series data between Insight Extensions and charts.
topMetric
Passes the top-by metric between Components.
unit
Passes a unit-metric pair between Components to specify in what units that metric is to be displayed.
value
Passes top limits between Components.
yAxisLimit
Passes the y-axis limit between Components.
zoomInterval
Passes the zoom time interval selected by a user between Components.
