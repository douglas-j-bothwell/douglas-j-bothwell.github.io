Deploying an Insight Extension
1)
Create an executable named run that is the main entry point into the Insight Extension.
2)
Place all the scripts and resources that are accessed by the Extension into a signed jar.
3)
Create a catalog with the insight that is to invoke the Insight Extension. When referring to the Insight Extension in the Insight you need to use the Insight name.
4)
Point to the jar file in the script field of the catalog.
5)
Point the update center at the catalog on a web server or the local insight directory to upload the Insight and its extension onto the box.
