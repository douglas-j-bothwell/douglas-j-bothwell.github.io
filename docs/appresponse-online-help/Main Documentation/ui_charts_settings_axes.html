Chart Settings: Axes
The following table describes the options available on the Chart Settings > Axes tabbed page. The specific options available in this page depend on the chart type.
 
Table 1
Chart Settings > “Axes” Tabbed Page
Option
Description
Metric
Select a metric to display in the Y axis (or X axis for Correlation charts).
Display Units
Select the units in which a metric is displayed. The unit options depend on the specific metric.
Auto
Uncheck Minimum and/or Maximum to specify a minimum or maximum value for the selected axis. If Auto is selected, the Minimum and Maximum are set to the lowest and highest values based on the selected metric and the correct project time.
Data Granularity
Select the time granularity of data shown in the axis (or both axes, depending on the chart type): 1 Minute, 5 Minute, 1 Hour, 1 Day. Defaults to Automatic.
Reference
Specify whether to display server-side or client-side metrics in the chart.
