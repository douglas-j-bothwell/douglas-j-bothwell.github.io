Group Table
Group tables (View > Table > [table_type]) are useful for showing metrics for many different group types.
Figure 8
Group Table
 
For descriptions of the available metrics, see Metric Definitions.
For descriptions of the drill-downs available in each table, see Group Drill-Down.
Director groups are available only in a domain and are documented in the domain chapter. For more information, see Aggregating Data to the Director.
A table cell shows the value of one metric (column) for one group (row) calculated over the current time selection. The cell is blank if the metric is not available for the group, or if no data for that group was collected during the project time window.
