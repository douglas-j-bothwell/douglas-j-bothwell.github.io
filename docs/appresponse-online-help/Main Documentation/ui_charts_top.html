Top Charts
Top Charts (View > Chart > Top Chart > [chart_type]) show the top groups for a specific metric. These differ from other charts in that the groups shown in these charts are selected automatically by AppResponse based on topping criteria specified in the chart itself.
The following Top Charts are available:
Top Multi-Group Time Series Chart—Similar to the Multi-Group Time Series Chart, but top groups are selected automatically.
Top Pie Chart—Similar to the Pie Chart, but top groups are selected automatically.
Procedure 1
Creating a Top Chart
1.
Create a new chart by choosing View > Chart > Top Chart > [chart_type].
2.
In a table or insight window, select an instance of the group type (such as Business Group, Application, or IP Conversation) you want to display in the Top Chart.
3.
Drag and drop the group into the Top Chart.
The chart shows the top groups of the selected group type, based on the topping criteria specified in the To: and By: toolbar controls. (If the specific group you dragged into the chart does not meet these criteria, it will not appear in the chart.)
4.
Specify the metric you want to display in the Y axis using the Select Metric toolbar menu.
5.
Specify the groups to show in the chart by setting the Top: and By: toolbar controls.
6.
Configure the chart as needed using the other available controls.
End of Procedure 1
