Testing the Insight
Although the Actual View of the Layout Grid shows the Insight as it will appear to the user, the Insight cannot query the database of the Appliance the Desktop Console is connected to, so its proper functioning cannot be checked. To display the Insight exactly as it will appear to the user and to connect to the database to test its functionality, click on Test/Layout in the lower right-hand corner of the Insight Development Kit window, under the Layout Grid, to display the Preview Insight Content window.
Figure 32
Preview Insight Content Window
 
Note: It is best practice to build an Insight on an Appliance and test it on a Director.
In this window you may operate the Insight against a data source to check its function. However, with one exception, no changes you make to the Insight (e.g., selecting a business group) are recorded.
The exception is size changes. If while in Test/Layout, you resize the entire window or any Visual Components within it, those changes are recorded when you save the Insight. This is a handy way to put the finishing touches on the appearance of an Insight and one of the best uses of Test/Layout.
Click Snapshot to save a .png file of the Insight to your hard drive.
Click Apply to apply any size changes and continue working in the preview window.
Click OK to close the preview window without making any changes.
Click Cancel to close the preview window without making any changes.
