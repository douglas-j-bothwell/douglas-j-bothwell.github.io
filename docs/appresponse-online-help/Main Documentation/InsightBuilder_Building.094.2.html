Building for Practice
This section show you how to build with certain Elements. The Insights shown below are not complex and would not be used as a solution, however the exercises here get you acquainted with certain Elements and how they work together.
Note: Remember to save often. There is no undo function. Rename Components as soon as you add them as it is important given that Interactions names are automatically generated from the names given to Components.
Using Selectors and Charts
This exercise shows you how to build an Insight that shows Throughput by Business Group.
Procedure 6
Building an Insight that shows Throughput by Business Group
1.
Choose File >New in the Insight Development Kit window.
2.
Choose View > Layout Grid > Actual.
3.
Click the Add Another Column button (along the right edge of the Layout Grid) to add another column to the layout.
Figure 107
Adding a Column to the Layout
 
4.
Fill the Active panes:
4.1.
Add a Business Group selector to the layout: Click the Business Group Selector toolbar button (near the Selectors: label in the toolbar) and click the left pane.
4.2.
In the treeview, right-click the Business Group Selector and rename it to Selector:BG.
4.3.
Click the Multi-Group Time Series Chart toolbar button (near the Charts: label in the toolbar) and click the right active pane.
4.4.
In the toolbar, right-click the Multi-group Time Series Chart to and rename it to Chart:multi-group.
Both panes are filled and renamed.
 
4.5.
Click Save and save the Insight as [name].xml in the Local Insight directory.
5.
Add the Group Interaction to pass the Business Group information to the chart.
5.1.
Switch to the Workflow view: choose View > Layout Grid > Workflow.
5.2.
Click Add Interaction.
The Components Interaction dialog opens.
5.3.
Configure the dialog box as follows:
• 
Type = Insight (this should be selected by default)
• 
Interaction = group
• 
Producer: Component = BG
• 
Consumer: Component = Chart:mult-group
 
5.4.
Click Apply.
5.5.
Click OK to close the Component Interactions dialog box and return to the main IDK window.
5.6.
Click Save.
6.
Test the Insight:
6.1.
Switch back to the Layout view by chooseing View > Layout Grid > Actual.
6.2.
Click Test/Layout.
The Preview Insight Content window appears.
6.3.
From the drilldown in the BGs, select a business group.
The chart populates.
6.4.
Resize the Insight to the desired dimensions.
6.5.
Click Apply and then OK to close the Preview window and return to the IDK.
The dimensions you specified in the Preview are applied to Layout grid.
6.6.
Click Save.
When you open this insight, it should look similar to this:
 
 
End of Procedure 6
 
Using Favorites Table and Drilldown Analysis
This exercise shows you how to create an Insight that populates a Favorites Table with the top five Business Groups by throughput and then sends the information to a Distribution chart.
Procedure 7
To begin: Using the Favorites Table and Drilldown Analysis
1.
Select File > New.
2.
Choose View > Layout Grid > Actual.
3.
Click the Add Another Column button (along the right edge of the Layout Grid) to add another column to the layout.
4.
Fill the Active panes.
4.1.
Click the Favorites Table toolbar button (near the Tables: label in the toolbar) and click the left Active pane.
4.2.
In the treeview pane, right-click the Favorites Table to and rename it to Table:1.
4.3.
Click the Distribution Chart toolbar button (near the Charts: label in the toolbar) and click the right Active pane.
4.4.
In the Elements pane, right-click the Distribution Chart and rename it to Chart:1.
4.5.
Click Save and save the Insight as [name].xml in the Local Insight directory.
5.
Add the Drilldown Analysis Component:
5.1.
Select Drilldown in the Analysis drop-down menu on the toolbar.
 
5.2.
Click Add Analysis.
The Set Analysis Parameters window opens.
5.3.
Select the Run Analysis on Startup checkbox.
With this option selected, the component will run its analysis as soon as the Insight is opened, without waiting for an interaction from another component.
5.4.
Click on the Drilldown: field and then click Choose Groups.
The Group/Drilldown Chooser appears.
5.5.
Select Business Group and choose any business group listed. This allows the business groups to be displayed in the Insight.
Figure 108
Group Drilldown Chooser
 
5.6.
Click OK to close the Group/Drilldown Chooser and return to the Set Analysis Parameters dialog box.
5.7.
Set the Top Limit to 5.
5.8.
Leave the Top Metric and Apply Analysis options as-is.
 
5.9.
Click OK to confirm your settings and return to the main IDK window.
5.10.
In the treeview, right-click the Drilldown1 analysis component to and rename it to DD.
6.
Switch over to the Workflow view by choosing View > Layout Grid > Workflow. Note the workflow layout shows three unconnected icons: one for the drill-down analysis component, one for the table, and one for the chart.
 
7.
Add two Group Interactions—one to send data for five Business Groups from the drilldown to the table (DD ==[data]==> Table:1) and another to send data from the table to the chart (Table:1 ==[data]==> Chart:1).
7.1.
Click Add Interaction.
The Components Interaction dialog box appears.
7.2.
If possible, move the dialog box so that you can see the treeview pane in the IDK window.
7.3.
In the Components Interaction dialog box, specify the following options:
• 
Type = Insight (this should be selected by default)
• 
Interaction = group
• 
Producer: Component = DD
• 
Consumer: Component = Table:1
7.4.
Click Apply.
In the treeview pane of the IDK window, a new Interaction object appears.
7.5.
In the Component Interactions dialog box, specify the following options:
• 
Type = Insight (this should be selected by default)
• 
Interaction = group
• 
Producer: Component = Table:1
• 
Consumer: Component = Chart:1
7.6.
Click Apply.
Another new Interaction element appears in the treeview.
7.7.
Click OK to close the Component Interactions dialog box and return to the IDK window. Note that the three objects are now connected in the Workflow layout.
 
7.8.
Click Save to save your work.
8.
Test the Insight.
8.1.
Click Test/Layout.
The Favorites Table populate with the top 5 by Throughput and sends the information to the Distribution Chart.
8.2.
Resize the Insight to the desired dimensions.
8.3.
Click Apply.
8.4.
Click OK.
The Insight is resized in the Insight Development Kit.
8.5.
Click Save.
When opened from the Local Insight directory, your Insight should look like this:
 
End of Procedure 7
 
 
Using the Workflow View
This exercise shows you how to take a Business Selector and a Favorites table and have the Favorites table populate with the top five BGs by throughput, as in Using Favorites Table and Drilldown Analysis, but using the Workflow view.
Procedure 8
Using the Workflow View
1.
Select File > New.
2.
Press CTL+W to change the Layout view to Workflow (this is equivalent to choosing View > Layout Grid > Workflow).
3.
Add a Favorites Table and a Distribution Chart by dragging the icons from the toolbar into the Layout Area.
4.
Rename the table and chart in the treeview:
4.1.
Right-click on the Favorites Table and rename it to Table:1.
4.2.
Right-click the Distribution Chart and rename it to Chart:1.
5.
Add the Analysis Component:
5.1.
Select Drilldown in the Analysis Type drop-down menu and click Add Analysis.
 
The Set Analysis Parameter dialog box appears.
5.2.
Set the options in this dialog box as follows:
• 
Run Analysis on Startup = selected (checked)
• 
Drilldown to: = Open the Group/Drilldown Chooser and select any Business Group
• 
Top Limit = specify a different value or leave the default
• 
Top Metric = leave default
• 
Apply Analysis = leave default
5.3.
Click OK to apply the Analysis Parameter settings and return to the IDK.
5.4.
In the Elements treeview, right-click on the new drill-down object and rename it to DD.
6.
The Visual Components are shown as Hidden in the Elements treeview. To move them out of Hidden Elements:
6.1.
In the Workflow toolbar (just above the workspace), click the Layout Grid toolbar button.
 
The Layout Components window appears.
6.2.
Drag the table and chart from the Hidden Components treeview to the Layout Grid, as shown below.
 
7.
Click OK to close the Layout Components window and return to the IDK.
8.
Add the Interactions.
8.1.
From the Workflow toolbar (immediately above the workspace), click the Add Interaction Mode toolbar button and select group from the Interaction pulll-down menu.
 
8.2.
In the workspace, select DD with the cursor and drag a line to Table:1.
8.3.
Select Table:1 with the cursor and drag a line to Chart:1.
 
9.
Test the Insight.
9.1.
Click Test/Layout.
The chart automatically populates with Business Groups in the Selector, and the top 5 by Throughput in the Favorites Table.
9.2.
Resize the Insight to the desired dimensions.
9.3.
Click Apply and then OK to apply the save the insight layout, close the Preview window, and return to the IDK.
9.4.
Press CTL+A to switch from the Workflow to the Actual layout view (this is equivalent to choosing View > Layout Grid > Actual).
The Insight is resized in the Insight Development Kit Actual view.
9.5.
Click Save and save the insight as [name].xml in the Local Insight directory.)
When opened from the Local Insight directory, your Insight should look similar to this:
 
End of Procedure 8
 
 
Using Toolbar and Top By Metric
This Insight exercise demonstrates you can use the toolbar to change the contents of a Favorites Table by metric.
Procedure 9
Using Toolbar and Top By Metric
1.
Select File > New.
2.
Click the “Add another row” button (at the bottom edge of the workspace) to add another row to the layout.
 
3.
Add a toolbar to the insight:
3.1.
Add a toolbar to the layout by clicking on the Toolbar button (next to the Utilities: label) and dragging into the top cell of the layout.
 
The Edit the Component dialog box appears.
3.2.
Click Add to open the Add Controls dialog box.
3.3.
Using Ctrl-click, select the Metric Pull-down Menu and Top Limit Input Field entries. Then click OK to return to the Edit the Component dialog box.
 
3.4.
In the Edit the Component dialog box, select the Top Limit Input Field entry and click Configure.
The Configure Toolbar Control dialog box appears.
3.5.
Enter 10 for the top limit and then click OK to return to the Edit the Component dialog box.
3.6.
Click OK to return to the IDK.
3.7.
In the Elements treeview, right-click on the Toolbar visual component and rename it to Toolbar:1.
4.
Add a chart to the insight.
4.1.
Click the Favorites Table toolbar button (near the Tables: label) and drag into the bottom cell in the layout.
 
4.2.
In the Elements treeview, right-click on the table component and rename it to Table:1.
4.3.
Click Test Layout and resize the layout as desired. Then click Apply and then OK.
Both panes are filled, properly named, and properly sized.
4.4.
Click Save and save the Insight as [name].xml in the Local Insight directory.)
5.
Add the Analysis Component.
5.1.
Select Drilldown in the Analysis pull-down menu (next to the Add Analysis button) on the toolbar.
5.2.
Click Add Analysis.
The Set Analysis Parameter window appears.
5.3.
Set the options in this dialog box as follows:
• 
Run Analysis on Startup = selected (checked)
• 
Drilldown to: = Open the Group/Drilldown Chooser and select any application in the Defined App field
 
• 
Top Limit = specify a different value or leave the default
• 
Top Metric = 10
• 
Apply Analysis = leave default
5.4.
Click OK to close the dialog box and return to the IDK.
5.5.
In the Elements treeview, right-click the Drilldown1 component to and rename it to DD1.
6.
Add Interactions to channel the desired data to the visual components:
6.1.
Click the Add Interaction toolbar button.
The Components Interaction dialog box appears.
6.2.
If possible, move the dialog box so that you can see the treeview pane in the IDK window.
7.
Add an interaction that sends the metric selected in the pull-down menu of the toolbar (Toolbar:1) to the drilldown (DD1).
7.1.
In the Components Interaction dialog box, specify the following options:
• 
Type = Insight (this should be selected by default)
• 
Interaction = metric
• 
Producer: Component = Toolbar:1
• 
Consumer: Component = DD1
7.2.
Click Apply.
8.
Add an interaction that sends the value specified in the “Top” field of the toolbar (Toolbar:1) to the drilldown (DD1).
8.1.
In the Components Interaction dialog box, specify the following options:
• 
Type = Insight (this should be selected by default)
• 
Interaction = value
• 
Producer: Component = Toolbar:1
• 
Consumer: Component = DD1
8.2.
Click Apply.
You have just created interactions that feed the “Top X” and Selected Metric selections from the toolbar to the drilldown.
9.
Add an interaction that feeds the Top X applications, based on the Selected Metric, from the drilldown to the table.
9.1.
In the Components Interaction dialog box, specify the following options:
• 
Type = Insight (this should be selected by default)
• 
Interaction = group
• 
Producer: Component = DD1
• 
Consumer: Component = Table:1
9.2.
Click Apply and then click OK to close the Components Interaction dialog box and return to the IDK.
The treeview pane of the IDK window shows the three new Interaction objects you just created.
10.
Test the Insight.
10.1.
Click Test/Layout.
10.2.
The chart automatically populates with the top 10 applications. Use the Metric drop-down list on the toolbar to change metrics.
10.3.
Click Apply.
10.4.
Click OK.
10.5.
Click Save to save your insight.
When opened from the Local Insight directory, your Insight looks like this:
 
End of Procedure 9
 
 
Using the Clear Interaction
This Insight exercise demonstrates how to use the Clear Interaction.
Procedure 10
Using the Clear Interaction
1.
Open the Insight created in Using Selectors and Charts.
2.
Add a row of Active panes.
3.
From Grid Edit Mode, click and drag BGs and Chart 1 to the Active pane below.
The Layout Area should look like the figure below.
Figure 109
Layout Area Before
 
4.
Add a Toolbar.
4.1.
Click the Toolbar Utility icon and click the left active pane.
The Edit Component window opens.
4.2.
Click Add.
The Add Control window opens.
4.3.
Using the CTRL key, select Clear and Horizontal Fill.
4.4.
Click OK.
4.5.
Using the Move buttons on the Add Control window, move Horizontal Fill above Clear.
4.6.
Click OK.
4.7.
Right-click the Toolbar and select Fill To Right.
4.8.
Click the Refresh control.
The Layout Area should look like the figure below.
Figure 110
Layout Area After
 
4.9.
In the Elements pane, right-click the Toolbar to and rename it to Clearbar.
5.
Open File > Save As and rename the Insight.
6.
Click OK.
7.
Add the Clear Interaction to clear information from the BGs and Chart 1.
7.1.
Click Add Interaction.
The Components Interaction dialog opens.
7.2.
Leave Insight as the Type.
7.3.
From the Interaction box, select Clear.
7.4.
From the Producer box, select Clearbar.
7.5.
From the Consumer box, using the Ctrl button, select Selector:BG and Chart:multi-group.
7.6.
Click Apply.
7.7.
Click OK.
7.8.
Click Save.
 
 
8.
Test the Insight.
8.1.
Click Test/Layout.
8.2.
From the drilldown in the BGs, select a business group.
The chart populates.
8.3.
Resize the Insight to the desired dimensions.
8.4.
Click Clear.
The Insight clears.
8.5.
Repeat steps b-d to test the Clear function.
8.6.
Click Apply
8.7.
Click OK.
The Insight is resized in the Insight Development Kit.
8.8.
Click Save.
When opened from the Local Insight directory, your Insight looks like:

Custom Insight

Using Tabbed Containers
This exercise show you how to use Tabbed Containers. The exercise uses the Workflow view.
Procedure 11
Using Tabbed Containers
1.
Select File >New.
2.
Press CTL+W to change the Layout Area to Workflow view.
3.
Add a Favorites Table, Multi-group Time Series Chart, Drilldown Analysis, Business Group Selector, and a Tabbed Container by dragging the icons from the toolbar into the Layout Area.
For the Drilldown Analysis:
• 
Check Start:.
• 
Change Group: to a Business Group.
• 
Enter 10 for Top Limit:.
• 
Leave Metric: and Group Index: as it is.
Figure 112
Set Analysis Parameters
 
4.
Rename the Components (remember that the visual components are in the Hidden Component folder).
4.1.
In the Elements pane, right-click the Favorites Table and rename it to FTable:1.
4.2.
In the Elements pane, right-click the Multi-group Time Series Chart and rename it to MGTSChart:1.
4.3.
In the Elements pane, right-click the Drilldown Analysis and rename it to Drilldown1.
4.4.
In the Elements pane, right-click the Business Group Selector and rename it to BG Selector:1.
4.5.
In the Elements pane, right-click the Tabbed Container and rename it to Container:1.
Figure 113
Rename the Components
 
5.
The Visual Components are shown as Hidden in the Elements pane. To move them out of Hidden Elements:
5.1.
From the Workflow toolbar, click the Layout Grid Button.
The Layout Component window opens.
5.2.
Drag the Visual Components from the Elements Pane on the Layout Component window to the Layout Grid, as shown below.
Figure 114
Layout Component Window
 
5.3.
Leave the BGSelector:1 and the FTable:1 as hidden (you will add them to the tabbed container).
6.
Add the Interactions.
6.1.
From the Workflow toolbar, click the Add Interaction Mode button and select group from the drop-down list.
6.2.
Select Drilldown1 with the cursor and drag a line to FTable:1.
6.3.
Select BGSelector:1 with the cursor and drag a line to MGTSChart:1.
6.4.
Select FTable:1 with the cursor and drag a line to the MGTChart:1.
 
 
7.
Check your progress. Your Element Pane should look like this:
Figure 115
Element Pane
 
8.
Add the Hidden Visual Components to the Tabbed Container:
8.1.
From the Element Pane, select and right-click BGSelector:1.
8.2.
From the right-click menu select Add to Tab>Container:1.
Figure 116
 
 
8.3.
When you select Container:1, the Tab Name dialog opens. Name the tab All Business Groups.
 
 
8.4.
Repeat steps a-c for FTable:1 and name the tab Top 10 Business Groups.
9.
Test the Insight.
9.1.
Click Test/Layout.
The chart automatically populates with Business Groups in the Selector, and the top 10 by Throughput in the Favorites Table.
9.2.
Resize the Insight to the desired dimensions.
9.3.
Click Apply
9.4.
Click OK.
9.5.
Press CTL+A.
The Insight is resized in the Insight Development Kit Actual view.
9.6.
Click Save. (Save the Insight as [name].xml in the Local Insight directory.)
When opened from the Local Insight directory, your Insight looks like:
Figure 117
Custom Insight Results
