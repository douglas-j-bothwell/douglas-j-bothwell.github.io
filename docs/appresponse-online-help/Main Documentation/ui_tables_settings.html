Table Settings
The Table Settings dialog box has the following tabbed pages for configuring an individual table:
Groups—Specify the topping criteria and the cell/row selection mode
Columns—Specify the metric columns to show and the sorting criteria
Toolbar—Specify the controls to show in the toolbar

Selection Mode
Specifies the selection mode for Shift-click and Ctrl-click operations when selecting and copying/pasting content from tables:
•
Multiple Rows—Select one or more rows using click, Shift-click, and Ctrl-click
•
Multiple Cells—Select one or more rows using click, Shift-click, and Ctrl-click
•
Single Row—Click in a row to select it (Shift-click and Ctrl-click disabled)
•
Single Cell—Click in an individual cell to selected it (Shift-click and Ctrl-click disabled)
The setting of Selection Mode is ignored for context menu operations (such as Mark for Export), when dragging and dropping the selection into another window, i.e,. you cannot use it to make selections for such operations.
