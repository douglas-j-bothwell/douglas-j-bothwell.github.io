Workspace and General Preferences
The Web Console workspace consists of a set of views arranged into tabs. The appliance retains a separate workspace for each individual user as long as the user is logged in. You can set the General Preferences to save and reload the workspace when you sign out and sign in later.
Figure 1
Workspaces for Individual Users, Saved on an Appliance
 
You can set the general preferences (workspace toolbar button, top right) to specify the initial layout the next time you sign in: you can open a specific view (or no view) or to save the workspace when you sign out.
Like workspaces, general preferences are saved for each individual user on the appliance. This means that you can retain the same state between login session (your workspace and preferences are not lost when you log out or close the browser window).

Home and Side Bar
The Sidebar shows all available views on the appliance. You can open/close this pane and set the General Preferences to specify where the Side Bar appears (in the Home tabbed page only or in all pages).
