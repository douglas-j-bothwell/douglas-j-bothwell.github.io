Layering Components
The Insight Development Kit offers three Components for changing Layers within an Insight.
Radio Button Layer Changer
HTML Layer Changer
Toggle Button Layer Changer
Radio Button Layer Changer
The Radio Layer Changer Component enables the user to toggle between any number of Layers in the Insight by clicking on the radio buttons.
Edit the Radio Button Layer Changer
Editing the Radio Button Layer Changer is a three-step process:
Figure 79
Edit Radio Button Layer Changer
 
This dialog box has the following components:
Caption—enter a caption that will appear above the radio buttons.
Layer Names—select the Layers that the Radio Layer Changer Component will control from a list of all the Layers in the Insight. There is no restriction on the number of Layers this Component can control.
Display Names—enter the text that should appear next to the radio button for each controlled Layer in a comma-separated list. There must be as many items in the comma-separated list as there are selected Layers in the list of controlled Layers, and they must be in the same order.
The Radio Layer Changer Component is neither a producer nor a consumer of Interactions.
HTML Layer Changer
The HTML Layer Changer Component allows the user to toggle between any number of Layers in the Insight by clicking on the links in the HTML. The links are of the form
<a href=”http://www.riverbed.com”>Riverbed</a>
If we refer to the portion of the URL with the “http:” as the protocol string, then the following protocol strings can be used in the links:
http: - launches an html document in an external browser
script: - launches a script in a separate window
guide: - launches an Insight saved in the local Insight directory.
guideManager: - launches any Insight in the Insight Manager that is saved on the Appliance
Edit the HTML Layer Changer
Editing the HTLM Layer Changer Component is a three-step process:
Figure 80
Edit HTML Layer Changer
 
This dialog box has the following components:
Layer Names—select the Layers that the HTML Layer Changer Component will control from a list of all the Layers in the Insight. There is no restriction on the number of Layers this Component can control.
Display Names—enter the text that should appear in the HTML link for each controlled Layer in a comma separated list. This text will be displayed automatically using a variable as specified in the next Layer. There must be as many items in the comma separated list as there are selected Layers in the list of controlled Layers, and they must be in the same order.
Enter the content of the Html document—define the HTML content. The links for the Layers will be created for you automatically if you enter the variables ${1}, ..., ${n} for each of the selected Layers. For example, to display the links for each Layer as an unordered list, use the following HTML:
<ul>
<li>${1}</li>
<li>${2}</li>
<li>${3}</li>
</ul>
WARNING: Do not make any HTML modifications until you have added all the Layers. If you modify the HTML and add another Layer, the modifications you have made are erased.
Interactions
The HTML Layer Changer Component is neither a producer nor a consumer of Interactions.
Toggle Button Layer Changer
The Toggle Button Layer Changer Component allows the user to toggle between two Layers in the Insight.
Edit Toggle Button Layer Changer
Editing the Toggle Button Layer Changer Component requires two steps:
Figure 81
Edit Toggle Button Layer Changer
 
This dialog has the following components:
Layer Names—select the Layers that the Component will control from a list of all the Layers in the Insight. You can only control two Layers with a Toggle Button Layer Changer Component.
Display Names—enter the text that should appear in the button for each of the two Layers, separated by a comma, in the order they appear in the Layer Names selector.
Interactions
The Button Layer Changer Component is neither a producer nor a consumer of Interactions.
