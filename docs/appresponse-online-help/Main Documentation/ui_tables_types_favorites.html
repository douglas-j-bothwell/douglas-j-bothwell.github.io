Favorites Table
The Favorites Table (View > Table > Favorites Table) provides functionality similar to bookmarks in a web browser: You can add any groups of interest to this table so that they are always easily accessible. You can also have multiple Favorites Tables open at the same time; each table is independent of the others. You can drag and drop groups into the table or use

Favorites Table
 
Add Group
To add a new group to the Favorites Table, click the Add Group toolbar button.
This opens the Group/Drilldown Chooser, which allows you to choose an arbitrary group to add to the table.
Figure 5
Group/Drilldown Chooser
 
The Group/Drilldown Chooser presents drilldowns in a manner similar to the a Group Table. . However, there are some subtleties to group selection in the group selection dialog that do not exist in the group tables. The following example describes how to add the following drilldown to a Favorites Table: HQ All > Connected Groups > UK Office > HTTPS (that is, “All HTTPS traffic between the business groups HQ All and UK Office”).
1)
In the Favorites Table, click the Add Group/Drilldown toolbar button. The Group/Drilldown Chooser opens.
2)
Drill down to:
     Business Group
Then select HQ All from the pull-down menu.
 
3)
Drill down further to:
     Business Group > Connected Groups > Business Group
Then select UK Office from the pull-down menu that appears.
 
4)
Drill down further to:
     Business Group > Connected Groups > Business Group > Applications
Then select HTTPS from the pull-down menu that appears. The treeview should now look like this:
 
5)
Click OK to close the Group/Drilldown Chooser and return to the Favorites Table. The following new row now appears in the table:
