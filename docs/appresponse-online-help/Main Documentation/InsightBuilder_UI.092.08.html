Working with Layers
Layers are analogous to pages in a book. Each time you create a new Layer you may choose a different set of Visual Components for the user to view, in effect creating a page in a book. A simple Insight has one Layer. Complex Insights may have multiple Layers with Components that allow the user to change the Layers, thus allowing the user to look through the pages in the book.
The Layer folders contain the Visual Components, Analysis Components, and Interactions in “Layer X” of the Insight. See Using Layers for information on building an Insight with Layers.
Please note the following points:
In this release, the default name for Layer in the UI is “Layer.” This is configurable by opening the Layer Control Edit box.
You can look at examples of Layers by opening File > Open Examples or File > Preview Examples.
–
Preview Examples shows an Insight with Layers.
–
Open Examples shows an Insight with Layers in the Insight Development Kit that is editable by the user.
Layer Modes
There are three modes for layering:
Replacement Mode
Progressive Mode
Hub and Spoke Mode
Replacement Mode
In Replacement mode, each Layer is independent of one another. There is no relation to the Layer before it, nor the Layer after it. You can switch to which ever Layer you wish at any time. All Components and Interactions are replaced whenever the Layer is changed.
Progressive Mode
In Progressive mode, each Layer builds on the Layer before it. Some constants may be shared between Layers. You cannot skip a Layer, but “travel” sequentially through Layer 1, Layer 2, etc. Some of the Components and Interactions can be shared between Layers.
Hub and Spoke Mode
The Hub and Spoke mode is a modified Progressive mode. There is a start Layer (hub) from which you can go to any other Layer (spoke) in the Insight. You cannot go from Layer to Layer with out going back through the start Layer. Every Layer is only one click away from the start Layer.
Layer Controls
Layer controls are how the user clicks through the different Layers in the Insight. Layer controls are located in the Insight Development Kit Toolbar. There are three types of Layer controls:
Radio Button Layer Changer
HTML Layer Changer
Toggle Button Layer Changer
Layer Toolbar Controls
When adding a toolbar to an Insight that is created with Layers, there are new toolbar options from the Toolbar Component. The Toolbar Component only shows the controls that are available for the mode of Layer you are using.
Combo Box - directly select any Layer.
Forward - click forward sequentially through the Layers.
Backward - click backward sequentially through the Layers.
Check Box - toggle between two progressive Layers. This is only applicable in the Progressive mode.
Buttons - toggle between two progressive Layers. This is only applicable in the Progressive mode.
Replacement mode allows Combo Box, Forward and Backward.
Progressive mode allows Combo Box (but you can only click sequentially), Forward, Backward, Check Box and Buttons.
Hub and Spoke mode allows Combo Box only.
