Web Transaction Analysis
You can use Web Transaction Analysis to
Track and analyze web page usage
Auto-discover downloads of web pages and related groups of web pages
Collect detailed metrics on web page usage, such as
–
Download times (full web page, all objects)
–
Overall web-page traffic
–
Total number of page downloads
–
Total number of slow pages
–
Distribution of HTTP response codes (percentage of 100, 200, 300, 400, and 500 codes)
Auto-decode and analyze SSL traffic
NOTE—AppResponse does not store, nor does it make accessible to end users, any decoded SSL-encrypted packets.
Auto-detect users (based on login information) and identify the web page downloads for each detected user
