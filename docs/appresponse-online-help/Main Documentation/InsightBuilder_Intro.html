Insight Development Kit (IDK) > Introduction
Introduction
This section supplies a brief overview of the design goals of the Insight Development Kit, how this manual is organized, and the system requirements for the Insight Development Kit.
It is assumed that the person using the Insight Development Kit has understanding of what information is available on the AppResponse appliance and how AppResponse works, especially drilldowns and top tables.
What Insight Development Kit Does
The AppResponse Insight Development Kit is a visual software development kit that enables you to create customized AppResponse Insights—XML-based workflow templates that walk users through the audit, baseline, troubleshoot, secure and optimize steps necessary to ensure that applications are running at their peak performance. You can use virtually any part of the extensive AppResponse user interface, plus a wide variety of supporting visual and data-processing elements, as interactive “widgets” to assemble a user interface customized to your customer's needs.
How This Manual Is Organized
This manual includes the following sections:
Insight Development Kit Basics—reviews the building blocks of an Insight and how they interact.
Insight Development Kit User Interface—reviews the fundamental features of the Insight Development Kit User Interface (UI), and lists some very important best practices that should be followed to avoid wasting your time making easily-avoided mistakes.
Components and Interactions—reviews, in detail, the various Elements available for constructing Insights.
Building Insights: Workflows and Exercises—leads you through exercises that provide a basic understanding of how to use Insight Development Kit. Depending on your learning style, it is possible that doing the exercises in this section before starting to read this manual will help you understand the other material better.
Insight Extensions and Scripts—describes how to create and run scripts installed on the appliance (Insight Extensions) and local scripts on the users’ machine.
Update Center—discusses the use of an Update Server for delivering new Insights to customers.
In addition to this manual, you should look at Creating Custom Insights: Introduction to Insight Development Kit in the AppResponse Tutorials section of the online documentation. This document provides a tutorial leading you through the steps necessary to design and build an Insight.
System Requirements
Insight Development Kit is an optional component of the Desktop Console. The system requirements for the Insight Development Kit are the same as for the Desktop Console.
