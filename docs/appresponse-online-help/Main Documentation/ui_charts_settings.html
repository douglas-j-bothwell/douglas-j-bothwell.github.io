Chart Settings
The Chart Settings dialog box enables you to configure all settings for a specific chart. To open this window, do one of the following:
Double-click in the chart display pane
Right-click in the chart legend and choose Settings
Click the Chart Settings toolbar button
The Chart Settings dialog is context-sensitive and offers different options for different chart types, as described in the following sections. Many of these options are also available through toolbar buttons or right-click menus.
Chart Settings: Axes
Chart Settings: Groups
Metrics—Offers the same controls as the Chart Settings: Groups tabbed page, but applied to metrics instead of Groups. It is available for Multi-Metric Times Series charts, including the Response Time Composition Chart. The assignments apply only to the current chart. The default colors are assigned automatically.
Chart Settings: Format
Legend—Offers many of the same controls as the chart legend context menu: legend placement and size, group name/information, and group path display. (These operations are described in Legend Right-Click Menu Operations.) This tab also offers Show Legend Background, which allows you to toggle back and forth between a background behind the legend or not.
Marker
Toolbar—show/hide the Chart Controls shown in a specific chart.
