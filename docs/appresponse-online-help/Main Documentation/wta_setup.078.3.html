User Data Collection: Setup
To collect user data in the Desktop Console, do the following:
1)
Specify the search criteria needed for AppResponse to auto-identify individual web clients and to correlate these clients with the web pages they download.
For more information, see Tracking Web Page Users: the User Session Tracking Manager.
2)
If you have not already done so, complete the steps described in Page Data Collection: Setup.
