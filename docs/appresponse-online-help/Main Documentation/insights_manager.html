Insight Manager
You can use the Insight Manager (Tools > Insight Manager) to browse and edit insights that are currently installed on the appliance. You can also delete insights, create new ones, and edit insights in the Insight Development Kit.
Figure 10-1 Insight Manager
 
Any Insight that you create on the Insight Manager is automatically uploaded to the tab you choose by using the Category drop-down list. Information entered in the Name and Description fields are seen on the tab in the Insight Index. Note—You must click Apply in order for the Insight to be uploaded.
In order to create or edit an Insight, you need to be familiar with the Insight Development Kit. For more information, see the Insight Development Kit User Guide for more information.
Creating a New Insight
Procedure 1
Creating a New Insight
1.
Click New.
The Default Insight Template window opens.
Figure 1
Default Insight Template Window
 
2.
Select from an existing template or Blank.
3.
Click OK.
The Insight Development Kit opens and you can edit the template or you can create something new.
End of Procedure 1
 
 
Editing an Existing Insight
You can edit an existing Insight in one of the following ways:
Select an editable Insight and click Edit.
The Insight Development Kit opens and you can edit the Insight.
Select an un-editable Insight, right-click and select duplicate to duplicate the Insight. Rename the duplicate and click Edit.
The Insight Development Kit opens and you can edit the Insight.
Note: The AppResponse default Insights are un-editable. To modify you must select duplicate.
Deleting an Insight
Click an Insight to see the full description. You may not edit an Insight, but you may delete it if desired by clicking Delete.
