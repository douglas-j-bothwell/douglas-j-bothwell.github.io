Insight Variables
Insight Variables allows you to pass key parameters and information (through embedding parameters in text or scripts) such as intervals and metrics that exist within the Desktop Console or an Insight.
The following Components can be used with Insight Variables:
Label Component
Static Text Component
Insight Launcher
Tab Container
Script Analysis
For instance, the following string:
Top $limit Groups by $metric for $group
creates the following text in a Label Component (given the corresponding values for the variables):
Top 10 Groups by Outbound Throughput for SAP Servers
Insight Variables take two notations:
$<interaction_name> (e.g., $group) when there is a space following the variable, and
${<interaction_name>} (e.g., ${group}) when there is no space following the variable.
CAUTION—When using Insight Variables in URLs, replace the dollar sign ($) with an underscore (_) or the Insight Variable does not function correctly.
Insight Variables are recognized for the following Interactions:
