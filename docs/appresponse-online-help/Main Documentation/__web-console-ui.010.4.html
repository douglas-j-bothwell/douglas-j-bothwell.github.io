Favorites Views
Favorites Views provide functionality similar to bookmarks in a web browser: you can collect and organize related data of interest and ensure that this data is immediately accessible. For example, you can create Favorites Views such as
My Sales Web App and Login Pages
All Remote Offices
Web Clients in East Coast Offices
Mission-Critical Apps for Users at HQ
You can add any groups of interest to a Favorites View so that they are always easily accessible. You can also have multiple Favorites Views open at the same time; each view is independent of the others.
Figure 5
Favorites View
 
To send data from a Navigator to a new or existing Favorites View, select the data and click
Send To > New Favorites View
Send To > [favorites-view-name]
 
To save the layout when you log out or close the window, set the View to Open preference to Open Workspace (Figure 2).
Note—It is good practice to rename your Favorites Views to something meaningful. To rename a view, right-click in the tab or click in the space between the title and the Close (X) button.
