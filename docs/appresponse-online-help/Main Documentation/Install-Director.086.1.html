Installing the Director
This chapter covers all information required to install the Domain Director. A domain is a set of AppResponse appliances that collect and analyze data cooperatively. The AppResponse Director is responsible for managing the appliances in the domain.
Pre-installation Information
Since the director does not collect data from the network, it does not need to be placed in the network at a specific location. Rather, it simply needs to be able to communicate with the appliances that make up the domain. As a result, most pre-installation considerations for AppResponse appliances (discussed in Pre-installation Information) are not necessary for the Director. You only need to review the communication requirements listed in Firewall Configuration.
