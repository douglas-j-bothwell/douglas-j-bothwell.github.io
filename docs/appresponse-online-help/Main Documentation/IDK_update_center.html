Update Center
The Update Center in the SteelCentral AppResponse Console allows you to automatically deliver Insights to whomever connects to your update server. An update server is simply a specified directory on a standard web server.
Figure 1 SteelCentral AppResponse Update Center
 
Using the Update Center to deliver Insights is quite simple. Briefly, the workflow is this:
1)
Create an FTP or SSH directory on a web server. This will be your update server.
2)
Create a Catalog of the Insights you wish to upload.
3)
Upload the Catalog and Insights to your update server.
When your update server is ready, your users can add it to the server list in their Update Center. They will be notified of new or updated Insights each time they connect to an appliance. (See the Update Center section of the SteelCentral AppResponse Operations Manual for details.)
Use this procedure to set up your update server.
Procedure 1 Setting Up an Update Server
1.
On the web server you want to use, create a directory with FTP or SSH write access.
2.
Place the Insights you want to distribute in a directory.
3.
If necessary, start the Insight Development Kit.
4.
Create a new Catalog file:
4.1.
Choose Tools > Create Catalog.
4.2.
Make sure the File Name field lists the path to the directory containing the Insights.
4.3.
Click Open.
A Catalog file named catalog.xml is created in the directory with the Insights.
5.
Open the Catalog Editor:
5.1.
Choose Tools > Edit Catalog.
5.2.
Make sure the File Name field lists the path to the directory containing the Catalog file.
5.3.
Click Open.
The Catalog Editor opens.
6.
Edit the Catalog and Insight properties:
6.1.
Click Properties and set the Required Version field. This is the minimum Console version that the Insights can be used with.
6.2.
Click OK.
6.3.
(Optional) Select each Insight and edit the properties as desired. Information about these properties can be found in Edit Catalog.
6.4.
Click OK.
7.
Upload the Catalog, Insights, and related files to the update server:
7.1.
Choose Tools > Upload Catalog.
7.2.
Set the source (original files) and destination (web server directory) information. Information about these fields can be found in Upload Catalog....
Note—Always use the Replace Catalog update method.
7.3.
Click Upload.
End of Procedure 1
