Using Layers
This Insight exercise shows the basic concept behind working with Layers using radio buttons and Progressive mode to show the top five applications in two different charts. It also shows how to add a Group Analysis and how to edit an Insight so that it doesnâ€™t automatically make a selection when populating.
See Working with Layers for more information.
Procedure 12
Using Layers
1.
Choose File > New to create a new Insight.
2.
Press Ctrl-A if you are now already in the Actual layout view.
3.
Create the first Layer:
3.1.
Create a new Insight with four Active panes.
3.2.
Click the Radio Button Layer toolbar button (near Layers: )and drag into the top-left pane in the layout.
If you do not see the Layers: toolbar buttons, choose View > Toolbars > Visual Components > Layers and select the checkbox.
3.3.
In the Elements pane, right-click on the LayerChanger and rename it to Layer:radio1.
3.4.
Click the Favorites Table icon and click the bottom-left pane.
3.5.
In the Elements pane, right-click the Application Selector and rename it to Table1.
3.6.
Double-click the Table1 cell.
The Edit the Component dialog box appears.
3.7.
Click the Table Settings toolbar button.
3.8.
In the Composition tab, verify that Automatic Selection is not selected.
3.9.
Click OK to close the Table Settings dialog box.
3.10.
Click OK to close the Edit the Component dialog box and return to the IDK.
3.11.
Click the Bar Chart toolbar button and click the lower-right active pane.
3.12.
In the Elements pane, right-click the Bar Chart to and rename it to Chart1.
3.13.
In the layout, right-click in the Radio Button cell and choose Fill To Right.
3.14.
Click the Refresh the Preview toolbar button (immediately above the layout).
3.15.
Resize the Layer:radio1 visual component to look like a toolbar.

Using Insight Modules with Tabbed Containers
This Insight exercise shows the basic concept behind working with Insight Modules and tabbed containers. In this exercise you will create a tabbed-container that has Business Group Selector and Multi-group Time Series chart in one tab, and Application Selector and Multi-group Time Series Chart in the other.
See Creating Insight Modules for more information.
Before you begin, you must set up a directory to store your Insight Modules, as described in Saving an Insight Module.
Procedure 13
Using Insight Modules with Tabbed Containers
Note: Begin by creating the Insight Modules:
1.
Save the first Insight Module (bg_modules):
1.1.
Select File > New.
The Layout Grid and Outline Area are ready to create a new Insight.
1.2.
Open File > Properties.
The Insight Properties window opens.
1.3.
Enter the name of the Insight Module as bg_modules in the Insight Name: dialog.
1.4.
Click OK.
Figure 123
Insight Properties Dialog Box
 
2.
Add two active panes.
 
 
3.
Fill the Active panes.
3.1.
Click the Business Group Selector icon and click the left active pane.
3.2.
In the Elements pane, right-click the Business Group Selector and rename it to BGs.
3.3.
Click the Multi-group Time Series Chart icon and click the right active pane.
3.4.
In the Elements pane, right-click the Multi-group Time Series Chart to and rename it to Chart 1.
Both panes are filled and renamed.
 
 
3.5.
Click Save and save the Insight Module in the directory youâ€™ve set up as bg_modules.imc. (See Saving an Insight Module for more information.)
4.
Add the Group Interaction to pass the Business Group information to the chart.
4.1.
Click Add Interaction.
The Components Interaction dialog opens.
4.2.
Leave Insight as the Type.
4.3.
From the Interaction box, select Group.
4.4.
From the Producer box, select BGs.
4.5.
From the Consumer box, select Chart 1.
4.6.
Click Apply.
4.7.
Click OK.
4.8.
Click Save.
5.
Add the Insight Module Interaction so the charts can use the Project Time on the Console.
5.1.
Click Add Interaction.
The Components Interaction dialog opens.
5.2.
Change the Type to Insight Module.
5.3.
From the Interaction box, select Interval.
5.4.
From the Producer box, select External Component.
5.5.
From the Consumer box, select Chart 1.
5.6.
Click Apply.
5.7.
Click OK.
5.8.
Click Save.
6.
Create the second Insight Module (app_module) by repeating steps 1-5, but add an Application Selector instead of a Business Group Selector and name the Insight Module app_module.imc.
7.
Create the Insight with the modules.
7.1.
Open File > New.
7.2.
Add the Tab Utility icon to the Active pane.
7.3.
Right click the Visual Components folder and select Add Hidden Visual Component.
The Select Component window opens.
7.4.
Open the Modules folder and select app_modules.
7.5.
Click OK.
 
 
7.6.
Repeat for bg_modules.
7.7.
Click Save. (Save the Insight as [name].xml in the Local Insight directory.)
8.
Add the Insight Modules to the tabbed container
8.1.
In the Elements pane, right-click Module: app_module.
8.2.
Select Add to Tab > Tab 1.
The Tab Name window opens.
8.3.
Write Applications and click OK.
8.4.
Repeat for Module: bg_module but name the tab Business Groups.
The Layout Area should look like the figure below.
 
 
9.
Test the Insight.
9.1.
Click Test/Layout.
9.2.
Select an application and the chart populates.
9.3.
Switch tabs and do the same for the business groups.
9.4.
Resize the Insight to the desired dimensions.
9.5.
Click Apply.
9.6.
Click OK.
The Insight is resized in the Insight Development Kit.
9.7.
Click Save.
When opened from the Local Insight directory, your Insight looks like:

Using Audio Clips and Help Pages
This Insight exercise shows the basic concept behind working with the Audio Clip Toolbar Control to sync with the Help Pages Toolbar Control. You will be using local files for both audio and help.
In this exercise you will use Using Toolbar and Top By Metric as a base to create an Insight with Audio Clips and Help pages.
Note: It is assumed that you know how to create .wav audio files and .html files and have saved them to your local machine in the Local Insights directory.
Procedure 14
Using Audio Clips and Help Pages
1.
From the Local Insight directory open the Insight you created when you built Using Toolbar and Top By Metric.
2.
Right-click on the Toolbar Component and select Edit Visual Component.
The Edit the Component window opens.
3.
Add new Toolbar Controls.
3.1.
Click Add.
The Add Control window opens.
3.2.
Using the CTRL button, select Audio Clips, Help Pages and Horizontal - Fill.
Figure 124
Add Control Window
 
3.3.
Click OK.
3.4.
Using the Move Down button, rearrange the controls to look like the figure below.
 
 
3.5.
Click OK.
The Toolbar should now look like the figure below.
 
 
4.
Configure the Audio File Clip.
4.1.
Right-click on the Toolbar Component and select Edit the Visual Component.
The Edit the Component window opens.
4.2.
Select Audio Clips and click Configure.
The Clip Configuration window opens.
4.3.
Click Add.
4.4.
In Clip: type $ and select $localUrl and then the path to the file.
Remember that $localUrl substitutes the path to the Local Insight directory. In the example shown below, the audio file is stored in the resources folder inside the Local Insights directory.
4.5.
In Clip Name: type the name you would like the clip to be called. Example: Help Track 1.
4.6.
Check Highlighting and Sync with Help.
4.7.
Click OK.
 
 
4.8.
For the sake of this exercise, configure another audio clip the same way with a different name.
4.9.
When you are done adding both, click OK on the Edit the Component window.
5.
Test the Insight by clicking Test/Layout.
6.
When the Insight opens, select Help Track 1 from Audio Clip drop-down list and click the play button.
The help file plays.
7.
Configure the Help Page.
7.1.
Right-click on the Toolbar Component and select Edit the Visual Component.
The Edit the Component window opens.
7.2.
Select Help Pages and click Configure.
The Help Configuration window opens.
7.3.
Click Add.
7.4.
In URL: type $ and select $localUrl and then the path to the file.
Remember that $localUrl substitutes the path to the Local Insight directory. In the example shown below, the audio file is stored in the resources folder inside the Local Insights directory.
7.5.
In Help Page Name: type the name you would like the clip to be called. Example: Help 1.
7.6.
Check Highlighting.
7.7.
Click OK.
 
 
7.8.
For the sake of this exercise, configure another help the same way with a different name.
7.9.
When you are done adding both, click OK on the Edit the Component window.
8.
Test the Insight by clicking Test/Layout.
9.
Click play on the Audio Clip control and the corresponding help page should open.
Note: Audio files and Help Pages correspond by how they are lined up in the drop-down list.
 
 
End of Procedure 14
