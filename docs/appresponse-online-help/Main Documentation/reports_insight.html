Insight Reports
An Insight report is a type of Desktop Console report. Whereas most console reports comprise a single table or chart, Insight reports can combine many of the elements found in Insights to provide a more comprehensive report. Insight reports combine multiple elements to provide a multi-layered view of specific elements in the network. The built-in console reports provided with AppResponse are Insight reports.
