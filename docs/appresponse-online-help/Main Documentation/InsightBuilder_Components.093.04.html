Using the Group Index
The Group Index may be set while editing a Component that is a consumer of the Group Interaction to enable that Component to pick a specific group out of the set of groups passed in the Interaction. The Group Index setting has no effect if only one group is passed in the Group Interaction.
Note: The Group Index is only available to a subset of Components.
The order of groups selected by the Group Index depends on the display order of the groups in the producing Component. In the case of a Table, this is from top to bottom; i.e., the top selected group is selected by setting the Group Index to “0” in the consuming Component, the next is selected by setting the Group Index to “1”, and so forth. This order may be ambiguous in multi-group Charts and other non-list-oriented Components.
