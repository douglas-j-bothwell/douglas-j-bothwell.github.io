Insight Development Kit Menus
Unlike many applications, the menu bar in the Insight Development Kit UI duplicates very few of the functions offered by the various toolbars. Instead, with the exception of the Edit menu, it either controls the appearance and function of the UI, or furnishes what may be called “housekeeping” functions associated with the overall function of the Insight Development Kit.
These various functions will be discussed in the context of their use. These are organized under File Menu, Edit Menu, View Menu and Tools Menu.
Insight Development Kit Toolbars
The various toolbars are used to add Insight Elements to the Insight, or, in the case of the Layout Grid toolbar, to control the function of the Layout Grid.
Outline Area
This area, which comprises the following three panes, displays textual and organizational information about the Insight under construction.
Element pane. Displays all of the Elements in the Insight under construction in a hierarchical tree. Each Layer in the Insight can be expanded to show all the Components and Interactions in it. Elements may be selected for editing via the Element pane context menu (right-click). Some Insight Development Kit functions can only be accomplished through the Element pane, such as adding Hidden Components.
For more information, see Working with the Element Pane; specific tasks are covered in the appropriate sections.
Description pane. Displays a brief description or explanation of the Element selected in the Element pane.
For more information, see Working with the Description Pane.
Layout pane. Allows a subset of layout parameters to be specified for the Element selected in the Element pane.
For more information, see Working with the Layout Pane.
Layout Grid
The Layout Grid displays the Insight under construction. There are three views to the Layout Grid:
Workflow View
Actual View
Schematic View
These Views are selected using the View Menu. Most of your time building an Insight is spent interacting with this element of the Insight Development Kit UI.
