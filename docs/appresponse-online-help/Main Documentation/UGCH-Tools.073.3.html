WHOIS Lookup Manager
The WHOIS lookup manager is used to query the WHOIS directory, a distributed database that identifies Internet domains and IP addresses. Invoke the WHOIS lookup manager from the Desktop Console Tools menu
Figure 5
WHOIS Lookup Manager
 
Issuing WHOIS Queries
The top of the whois lookup manager displays the whois servers that can be queried using the tool. Select the desired whois server from the list. From the Whois lookup tab, enter an IP address (e.g., “192.168.1.1”) or AS number (e.g., “AS1”) into the text field and hit the Query button. The results of the query to the selected whois server are displayed in the text field at the bottom of the whois lookup manager.
Configuring a New WHOIS Server
The “Whois parameters” tab can be used to add new whois servers to the manager.
Figure 6

Only the WHOIS server name and port number are required fields. The remaining fields are simply used to highlight results in WHOIS queries sent to these servers. The results that match these fields are displayed in red in the whois lookup tab.
