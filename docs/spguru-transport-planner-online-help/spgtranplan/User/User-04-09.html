Network Properties

Every network scenario has default settings that apply to the entire network-design workflow. You can view and change these settings in the Network Properties dialog box (Network > Network Properties). You must specify these settings before you apply network design operations such as routing, grooming, and dimensioning. For references to information about these settings, see Network Properties Dialog Box.
