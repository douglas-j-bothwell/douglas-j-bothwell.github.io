Configuring Individual Objects

In some cases, you might want to override the general equipment settings for specific objects. The object browsers (for nodes, links, connections, and rings) support operations for setting the properties of individual objects. Generally, you can right-click on an object in the browser treeview and choose an operation for the menu.
