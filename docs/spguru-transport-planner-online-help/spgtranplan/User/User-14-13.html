Summary View

The Summary view shows the total amount of equipment in the network using a tree structure. Table 13-9-Hardware Browser (Summary): Treeview lists the fields that are shown in the Summary view.

Figure 13-8 Hardware Browser (Summary View)

Table 13-9 Hardware Browser (Summary): Treeview 
Level
Object
Fields
Comments
1
Equipment Type
The browser shows the following information for each equipment type:
  • Type—SP Guru Transport Planner equipment type
  • Devices—Number of devices of this type in the network
  • Cards—Number of cards in all devices of this type
  • % Slots—Slot utilization on devices of this type
  • % Ports—Port utilization on the cards
  • Cost—Total cost of all devices and cards of this equipment type
  • % Cost—Cost of the equipment type as a percentage of the total network cost
 
2
Device
The browser shows the following information for each device:
  • Device—Device type
  • Quantity—Number of devices of this type in the network
  • Cards—Number of cards on devices of this type
  • Device Cost—Cost of all devices of this type
  • Total Cost—Total cost of all devices of this type and their cards
  • % Cost—Cost of this device type as a percentage of the total cost of all devices of the equipment type
 
3
Card
The browser shows the following information for each card:
  • Card—Card type
  • Quantity—Number of cards of this type in all devices of the specified type
  • Cost—Total cost of all cards of this type
  • % Cost—Cost of this card type as a percentage of the cost of all cards for the specified device type
 
End of Table 13-9
