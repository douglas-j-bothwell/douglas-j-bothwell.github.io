OTS Connectivity Information

The Node Connectivity (OTS Layer) window shows the minimum, maximum, and mean node connectivity as well as a matrix with the node connectivity between every node pair. To open this window, choose Info > OTS Layer Connectivity > Node Connectivity.

The Link Connectivity (OTS Layer) window shows the minimum, maximum, and mean link connectivity as well as a matrix with the link connectivity between every node pair. To open this window, choose Info > OTS Layer Connectivity > Link Connectivity.
