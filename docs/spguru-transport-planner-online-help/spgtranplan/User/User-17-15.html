Evaluate Availability: Batch Options

Table 16-11 lists the options you can specify for an Evaluate Availability command. For more information, see Failure Evaluation .

Table 16-11 Evaluating Availability: Batch Options 
Option
Description
Comment
-layer
Layer:
  • DCL = digital client layer
  • OCH = optical channel layer
DCL is the only applicable option if a network contains all EEC nodes (no OCC or EOCC nodes)
-load
Traffic matrix subject to evaluate availability
 
-opdf
Evaluate double failures:
  • 0 = neglect
  • 1 = calculate unaffected connections
  • 2 = apply restoration algorithm
For shared path protection or mesh restoration only
-optf
Evaluate triple failures:
  • 0 = neglect
  • 1 = calculate unaffected connections
For shared path protection or mesh restoration only
End of Table 16-11
