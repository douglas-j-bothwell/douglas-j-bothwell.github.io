Web Browser Requirements

The help system uses HTML frames and JavaScript to display topics. If you are unable to view or navigate to topics, make sure that your web browser meets the following minimum requirements.

  • Use Internet Explorer 5.0 or higher, Netscape 7.0 or higher, or Mozilla Firefox 1.06 or higher. Other web browsers may work, however, they are not supported.
  • Set your browser to always accept cookies.
  • If the text is too small, increase the browser's default font size.
  • To use a different font for the help text, change your browser's default font.
  • If text or graphics become unclear or distorted, use the browser's Refresh option.

Note—All help system features (including the display of the current topic and related topics in PDF using Adobe Acrobat) function correctly with the default settings of the supported browsers. If you experience unexpected behavior with the help system, restore the browser's default settings.
