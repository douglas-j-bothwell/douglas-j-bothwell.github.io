Defining a Multi-Layer IP-Over-WDM Network

Overview

A pan-European service provider has deployed an IP network on top of a DWDM transport network. The provider is on the verge of acquiring a new client who requires IP connectivity between three European sites. The provider wants to know what upgrades are needed for the network in order to meet the new client's requirements.

Objectives

  1. Define the relationships between a Layer-2/3 network in SP Guru Network Planner and a DWDM network in SP Guru Transport Planner.
  2. Dimension the Transport network to support the IP layer requirements.
  3. Inspect the routes taken by IP flows in the optical layer.
  4. Verify the network's resilience against single fiber breaks in the optical network.
