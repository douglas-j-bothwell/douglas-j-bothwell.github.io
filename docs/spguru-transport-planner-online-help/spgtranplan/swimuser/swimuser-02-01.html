SWIM Overview

The SWIM (SP Guru with Interlayer Modeling) product suite enables you to create and analyze network designs that integrate a layer-2/3 network (one that uses IP, ATM, or other Layer-2/3 technologies) with an underlying transport network. This chapter provides an overview of IP-over-transport networks and a description of the workflow for designing these networks using SWIM.
