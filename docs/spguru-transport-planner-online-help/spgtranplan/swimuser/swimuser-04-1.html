Creating a Multi-Layered Network Model

In the context of SWIM, a multi-layered network model consists of the following components:

  • A Layer-2/3 network scenario modelled in SP Guru Network Planner
  • A transport-layer network scenario modelled in SP Guru Transport Planner
  • A SWIM file that defines the relationships between links and nodes in the two scenarios

After you create a multi-layered network, you can design and analyze each layer using any of the standard operations in SP Guru Network Planner and SP Guru Transport Planner. You can also run integrated multi-layered operations available in SWIM; these operations are described in Menu Reference.
