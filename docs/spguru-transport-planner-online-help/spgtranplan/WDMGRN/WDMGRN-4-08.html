Link Usage Thresholds

You can now specify link usage thresholds on OCH and DCL links. This threshold is used to limit the number of resources (DCL timeslots or OCH wavelengths) that can be used during a routing, dimensioning, and grooming operation.

You can specify link usage thresholds manually (Network > Link Usage Thresholds); you can also import thresholds from data files. For more information, see the following sections of the SP Guru Transport Planner User Guide:

  • Link Usage Thresholds Data File
