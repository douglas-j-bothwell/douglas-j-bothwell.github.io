PL5 Enhancements

The following section describes a change introduced in release 12.0 PL5:

  • Explicit Delay Modeling of Links

Explicit Delay Modeling of Links

The delay on a link is now no longer strictly dependent on the link length. You can now set specific link delay values at any network layer, either in the Link Browser or by importing data files. This enables you to model end-to-end traffic delays more precisely: at any layer, you can specify link delays explicitly or have the delays calculated (based on the fiber length or on delay values in the underlying layer).
