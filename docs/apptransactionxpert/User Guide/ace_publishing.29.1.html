Publishing Reports
After diagnosing an issue with AppTransaction Xpert, you can generate reports of your findings for key stakeholders, including application developers, network engineers, and management.
AppTransaction Xpert provides the following reports:
•
Web Reports (HTML Format)
•
Microsoft Word Reports (RTF Format)
•
Difference Reports
•
Comparison Reports (RTF Format)
•
Webpage Performance Report (RTF Format)
•
Exporting AppDoctor Data to Spreadsheets (CSV Format)
•
Multiple-Transaction Reports (RTF Format)
(not available in all solutions)
•
Exporting Tree View Statistics
Choose Reports > Export Tree View Statistics… to export selected statistics from the Tree View window to a spreadsheet.
•
Exporting Window to Bitmap
Choose Reports > Export Window to Bitmap to generate a bitmap of the current window.
