Importing WAN-Accelerated Capture Data
If your capture data contains WAN-accelerated traffic, you might need to specify additional information about the accelerators, capture locations, and traffic. This section describes how to specify these settings correctly and how to verify and troubleshoot WAN-accelerated imports.
Note—Although many of these settings are auto-detected, in some cases these represent best guesses based on the available information. Therefore, it is good practice to verify the following settings.
Note—Before importing WAN-accelerated traffic, you should print a copy of the WAN Acceleration Worksheet, fill it out, and keep it as a reference.
