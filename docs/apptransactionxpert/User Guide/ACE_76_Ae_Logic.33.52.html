write_t()
Abstract
Writes a (time, value) pair to the specified statistic. Both the value and time are specified as arguments.
Class
Stat
Syntax
stat.write_t(time, value)
 
Argument
Type
Description
value
Double
Value to write to the specified statistic.
time
Double
Value to be used as the abscissa of the output vector’s new entry.
Return Type
 
None.
Example
authentication_stat_handle.write_t
((current_time - self.authentication_start_time), task_completion_time)
Details
This function appends a new (time, value) pair to an output vector for the specified statistic.
