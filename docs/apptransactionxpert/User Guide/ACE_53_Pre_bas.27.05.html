Publishing Results
You can publish QuickPredict results in a web report (Reports > Generate Web Report…) or a Microsoft Word report (Reports > Generate MS Word Report (.rtf)…). These reports include all open graphs and the parameter settings for each network path.
To export results to a spreadsheet program, such as Microsoft Excel, right-click in a graph and choose “Export Graph Data to Spreadsheet”.
