get_connection()
Abstract
Returns the connection in the tier pair with a given index.
Class
Tier_Pair Class
Syntax
tier_pair.get_connection(index)
 
Argument
Type
Description
index
int
Index of the connection of interest.
Return Type
Connection
Requested connection.
Details
index must be an integer between 0 and tier_pair.get_num_connections() - 1, inclusive. Other values will throw an IndexError exception.
