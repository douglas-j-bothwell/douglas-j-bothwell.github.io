add_label()
Abstract
Adds a label to this message with given rollup, short, and long text strings.
Class
Message Class
Syntax
message.add_label(rollup_text, short_text, long_text)
 
Argument
Type
Description
rollup_text
string
Text displayed in the visualization bubble when many messages overlap, along with the number of messages (for example, “10x: <rollup_text>”).
short_text
string
Text displayed in the visualization bubble for a single message.
long_text
string
Text displayed in the tooltip when the cursor hovers over a visualization bubble.
Return Type
 
No return value.
Details
Call task.redraw() to display these changes in the UI.
