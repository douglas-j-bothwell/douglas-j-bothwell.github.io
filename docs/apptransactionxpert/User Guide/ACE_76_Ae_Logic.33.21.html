get_tier_names()
Abstract
Gets a list of all logical tier names.
Class
Application_Base
Syntax
self.get_tier_names()
 
Argument
Type
Description
 
 
no arguments
Return Type
List of Strings
Tier names.
Details
When called for a parent task, this method returns a list of all logical tier names in the task, such as “client”, “web_server”, or “app_server”. When called for a child task, only the tiers in the child task are included in the list.
