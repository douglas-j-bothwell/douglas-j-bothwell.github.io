Using the Documentation Effectively
Product documentation is provided in an online HTML-based help system. The HTML-based help system includes the following features:
•
Treeview Hierarchy of Documentation—represents the organization of the documentation and allows for quick access to specific topics.
For more information, see Using the Contents Tab.
•
Global Index—provides hyperlinks to specific topics listed alphabetically.
For more information, see Using the Index.
•
Search Capability—allows you to search the documentation set for specific keywords.
For more information, see Searching the Documentation.
Note—Before running online help for the first time, be sure that the browser_prog preference is set with the full path of the HTML browser. To set the preference, choose Edit > Preferences and search for the preference.
Using the Help Window
The help window includes two frames:
•
Navigation Frame (left side)—Contains tabs for organizing links to topics.
•
Topic Frame (right side)—Displays topic text and navigation buttons/links.
Navigation Frame
The Navigation frame includes the following tabs for navigating to topics:
 
Tab
Description
Contents
Displays the help’s contents.
For more information, see Using the Contents Tab.
Index
Displays the help’s index.
For more information, see Using the Index.
Search
Displays search tool to look for topics containing specific text.
For more information, see Searching the Documentation.
To display a help topic in the Topic frame, click a link in the Navigation frame. See Topic Frame.
To resize the frames, drag the vertical border between the Navigation frame and the Topic frame.
Topic Frame
The Topic frame displays the topic selected in the Navigation Frame and includes the following buttons:
 
Button
Name
Description
Show in Contents
Displays the current topic in the Contents.
Previous
Displays the previous topic in help.
Next
Displays the next topic.
Home
Displays the help system home page. The home page includes quick links to the most commonly requested topics.
Related Topics
Displays a list of topics that are related to the current topic.
PDF
Displays the current topic and related topics in PDF using Adobe Reader. Use this option to print the current topic along with its related topics. (The PDF documentation does not include hyperlinks.)
Note—Adobe Reader must be installed to display topics in PDF.
Print
Prints the currently displayed topic.
In addition to the buttons, the Topic frame includes the following navigation links:
•
Breadcrumb trail—A breadcrumb trail appears between the Topic frame buttons and the topic text. The breadcrumb trail shows the current topic in relation to the topic hierarchy. Click on a breadcrumb topic link to display the corresponding topic.
•
Link to help system home page—At the end of each topic, there is a link to the first page of the help system.
Using the Contents Tab
The Contents tab displays a treeview hierarchy of the online documentation. Navigate the treeview hierarchy to drill-down to specific topics.
Procedure 3-1
Finding a Topic using the Contents Tab
1
Click the Contents tab at the top of the Navigation Frame.
Chapter title links appear in the Navigation frame.
2
Click a title in the Navigation frame.
The corresponding topic appears in the Topic Frame.
End of Procedure 3-1
 
 
Using the Index
The index contains links to topics, by keywords.
Procedure 3-2
Finding a Topic using the Index
1
Click the Index tab at the top of the Navigation Frame.
2
Select the letter at the top to display index entries starting with the selected letter.
3
Select the index entry of interest.
The corresponding topic appears in the Topic Frame.
End of Procedure 3-2
 
 
Searching the Documentation
Use Search to find all topics that contain specific words.
Procedure 3-3
Finding a Topic using the Search
1
Click the Search tab at the top of the Navigation Frame.
The Search text and scope boxes appear in the Navigation frame.
2
Type the words that you want to find.
Select the scope of the search. Select “All Available Books”, a book, or a specific section of a book.
3
Click GO.
Links to topics that contain the search words appear in the Navigation frame.
Topics are organized by topic hierarchy. Each result is preceded by a ranking percentage.
4
Select the topic that you want to view.
The selected topic appears in the Topic Frame.
End of Procedure 3-3
 
 
Web Browser Requirements
The online product documentation uses HTML frames and JavaScript to display topics. If you are unable to view or navigate to topics, make sure that your web browser meets the following minimum requirements.
•
Use Internet Explorer 6.0 or higher or Mozilla Firefox 3.5 or higher. Other web browsers may work, however, they have not been tested.
•
Set your browser to always accept cookies.
•
If the text is too small, increase the browser’s default font size.
•
If text or graphics become unclear or distorted, use the browser’s Refresh option.
•
All online product documentation features (including the display of the current topic and related topics in PDF using Adobe Acrobat) function correctly with the default settings of the tested browsers. If you experience unexpected behavior, restore the browser’s default settings.
Note—For solution-specific web browser requirements, see the Support Website.
