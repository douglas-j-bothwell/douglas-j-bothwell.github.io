System Environment
The AppTransaction Xpert directory structure allows for files of different types to be stored separately, which allows for smooth upgrades.
AppTransaction Xpert files are organized into the following directories:
•
Installation Directory (<install_dir>) contains system software:
—
System Directory (sys)
—
Documentation Directory (doc)
•
User Home Directories include configuration and by-product files:
—
Administration Directory (op_admin)
—
Model Directory (op_models)
—
User Report Directory (op_reports)
