Managed Servers
Specifies a space-separated list of <server>, <port> pairs to be explicitly contacted by the License Manager. This preference causes license servers that are not in the same IP network segment as the License Manager to appear in the license server treeview. If <port> is not specified, “port_a” is assumed. For example: iasimov,port_a rbradbury,port_b aclarke10.
 
Tag
license_managed_servers
Data Type
string list
Default Value
<empty>
