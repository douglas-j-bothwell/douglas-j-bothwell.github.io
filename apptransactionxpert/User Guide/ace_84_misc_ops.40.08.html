Advanced Menu
The following operations are accessed via the Advanced menu.
•
Example Scripts
Displays a submenu of built-in scripts. Select a script from the submenu to run it.
•
New Icon Database… / Open Icon Database…
Opens a new/existing icon database. For more information, see Icon Database Editor.
