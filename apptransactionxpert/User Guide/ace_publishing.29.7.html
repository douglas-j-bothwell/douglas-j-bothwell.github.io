Exporting AppDoctor Data to Spreadsheets (CSV Format)
To export AppDoctor data to a tab-delineated text file, choose Reports > Export AppDoctor Statistics. You can then view AppDoctor results using a spreadsheet program like Microsoft Excel. The resulting file includes diagnoses and raw values from the AppDoctor Diagnosis window, and statistics from the AppDoctor Statistics window.
