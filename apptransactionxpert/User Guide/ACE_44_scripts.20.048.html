get_parent_dependency()
Abstract
Returns the parent Dependency object for this message.
Class
Message Class
Syntax
message.get_parent_dependency()
 
Argument
Type
Description
 
 
no arguments
Return Type
Dependency
Parent Dependency object; “None” if the message is either the first application message or is acausal.
Details
None.
