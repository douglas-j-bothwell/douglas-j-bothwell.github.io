get_captured_size()
Abstract
Returns the captured size of this network packet.
Class
Network_Packet Class
Syntax
network_packet.get_capture_size()
 
Argument
Type
Description
 
 
no arguments
Return Type
int
Captured size of packet, in bytes.
Example
total_size = 0.0
conn = current_tier.get_connection (conn_index)
for packet in conn.get_network_packets():
   total_size += packet.get_captured_size()
avg_size = total_size / conn.get_num_network_packets()
Details
None.
