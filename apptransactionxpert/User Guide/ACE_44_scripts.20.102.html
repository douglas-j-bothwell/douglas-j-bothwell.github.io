get_tier_pair()
Abstract
Returns the tier pair in the task with a given index.
Class
Task Class
Syntax
task.get_tier_pair(index)
 
Argument
Type
Description
index
int
Index of desired tier pair.
Return Type
Tier_Pair
Requested tier pair.
Details
index must be an integer between 0 and task.get_num_tier_pairs() - 1, inclusive. Other values will throw an IndexError exception.
