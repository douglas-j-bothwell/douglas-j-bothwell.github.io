delete_others()
Abstract
Permanently deletes components not in a given list.
Class
Task Class
Syntax
task.delete_others(keep_list)
 
Argument
Type
Description
keep_list
List of Component objects
List of components to be retained.
Return Type
 
No return value.
Details
keep_list can be of mixed component sub-types (such as Transactions, Messages, and Network Packets). Because a GUI recalculation is performed after each call to this method, you should ideally build the full list and call this method once.
