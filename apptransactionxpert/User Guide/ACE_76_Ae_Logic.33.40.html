get_attr()
Abstract
Gets the value of an attribute of a network model node.
Class
Node
Syntax
node.get_attr(attr_name, default_value)
 
Argument
Type
Description
attr_name
String
Fully qualified name of the attribute being queried (see Details).
default_value
Double, Integer, or String
Value to be returned if the named attribute is not present in the Node object. The type of the default value should match the type of the node attribute.
Return Type
 
Value of the attribute, if found; otherwise default_value.
Example
client_node = self.get_tier_node('client')
if client_node.get_attr('Client Address', 'Unknown') == 'new_york':
client_node.set_attr('Client Address', 'big_apple')
Details
attr_name must uniquely identify an attribute found on the node, including the full hierarchy for sub-attributes. If the value of attr_name is “name”, the hierarchical name of the node will be returned instead of the value of the attribute.
In QuickPredict, this method always returns default_value (because there are no nodes in this context).
