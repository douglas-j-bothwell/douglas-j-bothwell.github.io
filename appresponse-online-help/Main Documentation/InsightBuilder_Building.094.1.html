Building Insights: Workflows and Exercises
The Insight Development Kit is a complex application. This chapter, which introduces you to some of the major concepts and features of the Insight Development Kit by guiding you through some exercises that can aid you in the creation of Insights.
We encourage you to work through this chapter carefully, as it is not really possible to learn how to build an Insight without doing so: just reading the reference material is not sufficient.
WARNING: It is assumed that the person using the Insight Development Kit has understanding of what information is available on the AppResponse appliance, and how the appliance works. Building Insights is an advanced operation and one should learn by building simple Insights and then progressing to more complex Insights.
This chapter is organized into the following sections:
Building for Practice—short exercises that acquaint you with certain elements in the Insight Development Kit.
Note—These exercise are just that: exercises. They are not necessarily the most efficient way to use the UI, nor the most practical. They are designed to introduce you to using the UI.
Advanced Exercises - exercises that acquaint you with some more advanced elements in the Insight Development Kit.
