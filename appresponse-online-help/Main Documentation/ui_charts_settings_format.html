Chart Settings: Format
The format tab offers control of gridlines, chart style, and the information panel. It also provides an important workflow function similar to Set as Project Time.
 
Table 1
Chart Settings > “Format” Tabbed Page
Option
Description
Gridlines
Show/hide and edit the colors / line styles for the X and Y axes
Chart Style
Same as the Chart Style toolbar button.
Show Border
Show/hide 3D border
Show/Hide Title
Show/hide chart title
Show Information Panel
Show/hide Information Panel at the bottom of the chart.
Auto-Zoom Select Time
If this box is checked, the project time selection is controlled by the chart’s Horizontal Zoom control. Same as the Set as Project Time operation (available by right-clicking in the chart legend).
