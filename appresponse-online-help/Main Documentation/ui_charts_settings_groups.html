Chart Settings: Groups
The Groups tab enables the display of summary statistics and trend line(s), and assigns colors to groups in a chart that can show multiple groups (Multi-Group Time Series, Correlation, Distribution, Cumulative).
 
Table 1
Chart Settings > “Groups” Tabbed Page
Option
Description
Show Summary Statistics
Shows average/minimum/maximum values of the time series displayed for each group in the legend.
Show Total Statistic
(Pie Charts only) Show the sum of all metrics shown in the chart.
Show Trend
Shows a trend line for each group displayed. Same as the Trend control.
Show Trend and Show Summary Statistics are not always available on the Group tab.
Sort by
(Multi-Group Time Series Charts only) Sort groups based on name or the displayed metric.
Normalization
Useful when you want to view the metric values as fractions or percentages of a higher (total or maximum) value. See Chart Settings: Groups: Normalization.
Color
Click a the color next to a metric name to change it.
