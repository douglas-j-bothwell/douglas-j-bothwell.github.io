Global Search in Web Console
The Web Console includes a Search field (top right) that enables you to search configuration and traffic data on the appliance. When you enter a search string and press Enter, a new Search page appears.
 
Note the following:
The initial time selection in this page is the most recent hour.
A new search field and Search button appear below the time slider. These allow you to do a new or revised search in the same window. To repeat the search on a different time window, for example, change the time selection and then click Search.
The search results in this page show group definitions that match the search string. The appliance searches the following group types and shows all definitions that match the search string:
–
IPs
–
Applications (Defined, Other, and Web)
–
Business Groups
–
Page Families
If a group has traffic within the current time selection, the search result also includes links to Web Console views for this traffic (Navigators, Quick Views, and Individual Page views).
Web Console searches show links to Web Console views only. To search Insights, Managers, and other Desktop Console views, go to the Desktop Console.
Supported Search Terms
The Web Console search in this release supports the following:
Search on multiple strings:
one two three => all entries with one | two | three
Search on strings to exclude:
one two -three => all entries with (one | two) & !three
Search on exact phrases:
“one two three" => all entries with the string "one two three"
Search on IP addresses and ranges (using subnets or ranges) as well as ports and port ranges:
192.168.14.2 => traffic for this IP (in the Related Data category) and all Business Groups with this IP
192.168.14.36-192.168.14.255 => all Business Groups with IPs in this range
92.168.14.2/16 => traffic for this IP (192.168.14.2) and all Business Groups that include IP addresses in this range (192.168.0.0-192.168.255.255)
8080 => all applications that use this port
8080-8100 => all applications that use ports within this range
Note—When you search on a range of IP addresses or port numbers, do not include any spaces in the search string (enter 1.1.1.1-2.2.2.2 and not 1.1.1.1 -2.2.2.2).
Search using a fully-qualified host or domain name. In addition to searching for the specified search string, AppResponse performs a DNS lookup and then searches on the resolved individual IP address or multiple IP addresses.
Search for web page families. Note the following:
–
To search for one or more page families, you must enter a search that is at least four characters long and includes either a period character ('.') or the string http.
–
You can use asterisks as wildcards to search for web page families only.
–
The search results for web page families are additionally sorted by the times when individual pages were downloaded (most recent downloads first).
