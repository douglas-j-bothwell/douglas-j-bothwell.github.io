Alerting and the Dashboard
The appliance includes sophisticated alerting capabilities to provide proactive notification of significant network events.
There are two types of user-configurable alerts:
Static Alerts
Static alerts are generated when the appliance observes a network condition that crosses a user-defined threshold. They can be defined against a group or set of groups and applications, and the threshold defined as Boolean operations on any metric or on a set of metrics.
Adaptive Alerts
The appliance includes a set of analyses, called adaptive alerts for automatically detecting problems. These alerts, based on sophisticated statistical techniques, make it much easier to identify issues on your network.
When an alert triggers, it can generate an SNMP trap, SYSLOG message and/or an email message. All triggered alerts are recorded in the Alert Log and displayed in the Problem Management Dashboard.
Alerts are configured in the Desktop Console using the Alert Manager.
