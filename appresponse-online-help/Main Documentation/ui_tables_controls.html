Edit Columns
Select the metrics to show or hide in the table. For detailed metric descriptions, see Metric Definitions.
Figure 10
“Edit Columns” Toolbar Button
 
Group/Drilldown Topping Limit
Specifies the number of Top N groups to show in the table. You cannot specify a top value higher than 3000.
Figure 11
“Group/Drilldown Topping Limit” Field
 
Group/Drilldown Topping Metric
The metric used to determine the Top N groups to show in the table.
Figure 12
“Group/Drilldown Topping Metric” Menu
 
Note: AppResponse collects metrics for the top groups based on total throughput, so selecting a different metric gives you the top N groups within the groups for which metrics have been collected. If AppResponse did not collect metrics for a Business Group or IP Conversation (for example) due to low total throughput, that group will not appear in the table.

For more information, see How AppResponse Xpert Selects Top Elements.
Restore Default Columns
Restore the default set of metrics shown in the table.
Figure 13
“Restore Default Columns” Toolbar Button
 
Row Sort
To sort a table based on a specific column, click the Row Sort control (dot or up/down arrow) in the corner of the column header. Clicking the arrow toggles the sort order.
Figure 14
“Row Sort” Control in Column Header
 
Select an Appliance (Director Only)
On a regular AppResponse appliance, the group table shows data collected by that appliance. In a domain, a group table in a project connected to the Domain Director shows data collected by appliances that belong to the domain. The appliance that serves as the source for this data is specified using the From control.
Figure 15
“Select Appliance” Menu
 
The console transparently connects to the selected appliances to retrieve the data. This might result in a small delay the first time AppResponse retrieves data from that appliance. When working with a project connected to an appliance, a separate project must be created to examine data from an alternate appliance.
Show Table Type
Sets the group type shown in a Group Table.

“Show Table Type” Menu
