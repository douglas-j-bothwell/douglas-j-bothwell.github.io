Creating Topology Views From Tables
The IP Topology window can be invoked directly from any table. To create the view, select a table row in the table with an IP address group then press the right-mouse button and select Open New, and then select IP Topology view. A new IP Topology view is created displaying the topology to the IP address selected. Select multiple rows in the before opening the IP topology view to see the topology to multiple IP addresses simultaneously.
AS topology views can also be invoked directly from tables. Select a row, or rows, representing AS groups (ISP AS, Peer AS or Dest AS), right-click, select Open New and then select AS Topology view.
