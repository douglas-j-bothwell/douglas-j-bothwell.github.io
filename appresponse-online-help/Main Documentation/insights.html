Insights
Insights are graphical action guides that guide you through the best practices for accomplishing a wide range of tasks:
Audit or discover what applications are running on the network
Baseline the actual usage and performance of applications on the network
Troubleshoot network infrastructure and application issues
Identify security violations and breaches
Optimize your application infrastructure: choose, test and manage an optimization solution
Customize your own Insights
A default set of Insights come pre-installed on the appliances and directors. Updated insights are available from the Update Center. From the Desktop Console they may be downloaded using the Update Center and managed through the Insight Manager. You can also integrate Insights into the Problem Management Dashboard by linking them with Static Alerts or Adaptive Alerts.
You can also build your own Insights and keep them in the local Insight directory, or upload them to the appliance. Insights are built with the Insight Development Kit. Information can be found in the Insight Development Kit Guide.
