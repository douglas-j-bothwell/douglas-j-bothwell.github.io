Page Data Collection: Setup
To collect web page data in the Desktop Console, do the following:
1)
If you plan to observe any web traffic that exchanges data in key-value pairs, configure AppResponse to mask out the private data.
It is good practice to do this before you start collecting page data. For more information, see Masking Private Data in Observed Web Traffic.
2)
If you plan to observe any web traffic that relies on SSL encryption, configure AppResponse to decode this traffic.
For more information, see SSL Decoding: Enabling and Configuring
3)
Specify the server IP addresses and ports for which you want to monitor traffic and then enable Web Transaction Analysis.
For more information, see Page Analysis: Enabling and Configuring.
