SLA Dashboard
The Service Level Agreement (SLA) Dashboard.is is for users who, at a quick glance, can look at the service level of specific applications and business groups. The service level agreement is monitored by events based on a configurable condition applied to a set of metrics and then counting the events to see if the agreement is satisfied. The SLA Dashboard is configured using the SLA Dashboard Manager.
To view the SLA Dashboard select View > SLA Dashboard > [dashboard_name]. SLA Dashboards are configured in the SLA Dashboard Manager.
Your can also view the SLA Dashboards on the web UI. For more information, see Viewing SLA Dashboards through the Web Interface.

SLA Dashboard Manager
The SLA Dashboard Manager is used to configure SLA Dashboards. To open the SLA Dashboard Manager, choose Tools > SLA Dashboard Manager…
Figure 28
SLA Dashboard Manager
 
Note: Remember that you must click OK on the SLA Dashboard Manager window after creating a new dashboard or editing an existing one. If you do not click OK, your changes are not saved.
Creating A New SLA Dashboard
Procedure 1
Creating a New SLA Dashboard
1.
From the SLA Dashboard Manager, click New.
The Edit SLA Dashboard Content window opens.
Figure 29
Edit SLA Dashboard Content Window, Global Tab
 
2.
On the Global tab, enter the following information:
• 
Title - Choose the name of the SLA Dashboard as seen in the console.
• 
Days - Select the days you want the dashboard to monitor the SLA.
• 
Time - Select the business hours on the days selected to monitor the SLA.
• 
Enter the percentage of time that the Default SLA condition must be met.
• 
Select the metric for the Default SLA and the time for response.
• 
Enter the warning level by percentage of the SLA violation.
3.
Open the Dashboard tab.
Figure 30
Edit SLA Dashboard Content window, Dashboard tab
 
4.
On the Dashboard tab, enter the following information:
• 
Row Height/Column Width - Configure the height and width of the dashboard in pixels.
• 
Using the style icon to show the alert level. You may choose from colored lights or the entire cell.
Figure 31
SLA Dashboard Render Style
 
• 
% - Set whether the percent values appear in the SLA Dashboard cells.
• 
Click the Add Applications icon () to add the applications you want monitored.
• 
Click the Add Business Group icon () to add the Business Groups.
To customize a specific cell, see Customizing a Specific SLA Dashboard Cell.
5.
Click OK.
You are returned to the SLA Dashboard Manager.
6.
In the Name field enter the name of the SLA Dashboard that is shown in the SLA Dashboard Manager information pane. (This is different than the name that is shown in the console).
7.
In the Description field enter a description for the SLA Dashboard.
8.
From the Category drop-down menu, select a category.
To create a new category, delete the name in the Category field and enter the name of the new category.
9.
Click OK.
End of Procedure 1
 
 
Customizing a Specific SLA Dashboard Cell
NOTE—You cannot specify more than 100 unique SLA thresholds per AppResponse appliance. The total number of unique theresholds is computed by adding the number of custom thresholds to the default threshold and summing over all SLA dashboards.
Procedure 2
Customizing a Specific SLA Dashboard Cell
1.
Select the cell to be customized.
2.
Change the SLA threshold default settings.
You can also check No SLA and the cell does not show data.
3.
Click OK or Apply.

Dashboard Tab Right-Click Menu
There are two right-click menus available on the Dashboard tab.
Row/Column right-click menu
Custom cell right-click menu
Procedure 3
Changing the Label of Applications and/or Business Groups
1.
Right-click in the desired heading cell (row or column).
Figure 33
SLA Manager, Dashboard Tab Right-Click Menu for Rows/Columns
 
2.
Select Edit Row Label....
The Edit Label dialog opens.
3.
Type the name of the new label.
Figure 34
SLA Manager, Dashboard Tab - Edit Label Dialog
 
4.
Click OK.
From the row/column right-click menu, you can also delete entire rows and columns by selecting Delete Row (when clicking in row) or Delete Column (when right-clicking in column).
The right-click menu for a customized cell allows you to do the following:
• 
Reset to Default - The custom setting is cleared and the cell now has the default SLA settings
• 
Apply Definition to Row - The custom SLA settings for that cell are applied to the entire row
• 
Apply Definition To Column - The custom SLA settings are applied to the entire column of cells.

Editing an Existing SLA Dashboard
Procedure 4
Editing an Existing SLA Dashboard:
1.
From the SLA Dashboard Manager, select an existing dashboard from the information pane.
2.
Click Edit.
You can now edit the dashboard.
Note: Ensure you click OK or Apply on the SLA Dashboard Manager in order for changes you made in the Global tab, Dashboard tab, or the SLA Dashboard Manager to be saved.
End of Procedure 4
 
 
Deleting an SLA Dashboard
Procedure 5
Deleting an SLA Dashboard
1.
From the SLA Dashboard Manager, select an existing dashboard from the information pane.
2.
Click Delete.
3.
Click OK if you are finished, or Apply if you want to perform another operation.
End of Procedure 5
