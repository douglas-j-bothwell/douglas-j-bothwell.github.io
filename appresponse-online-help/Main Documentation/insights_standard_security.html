Security Insights
Security Insights help you detect, isolate, and correct security problems on your network. They can pinpoint worms, spam zombies, and other malware, and identify rogue traffic.
Worm Hunt is the only Security Insight.
Worm Hunt
Worm Hunt is located in the Secure tab. This Insight finds worms running on your network (or tries to) without the need for programmed signatures. It can do this because the majority worms behave the same: once they infect a computer, they start scanning the network looking for other vulnerable machines. But because they don’t know the structure of the network, most of the time they’re trying to connect to computers that don’t exist, which generates failed connections (SYNs without corresponding SYN-ACKs.
This insight includes a “Drill into:” pull-down menu that enables you to
Drill into Apps (Selected Business Group > Top Applications >
Top IPs > Top Connected IPs)
This view is more useful for viewing and analyzing top applications within a particular Business Group.
Drill into IPs (Selected Business Group > Top IPs >
Top Connected IPs > Top Applications)
This view is more useful for viewing and analyzing top individual IP addresses within a selected Business Group.

Procedure 1
Using the Worm Hunt Insight
1.
Select from Top Business Groups ranked by failed connections or from All Business Groups.
2.
Zero in on a spike in throughput by “click sweeping” that part of the chart by left-clicking and holding the mouse to the left of the spike, sweep across it, and release at the right side of the spike. A gray column overlays the spike.
3.
In the Application pane (lower left), select an application with a high number of failed connections in order to identify the worm.
4.
In the Top IP pane (lower middle) you can see which addresses are infected by the worm. Select one to see which addresses are under attack.
5.
In the Top 10 Connected IP pane (lower right) you can see the addresses that the worm has attempted to infect.
End of Procedure 1
