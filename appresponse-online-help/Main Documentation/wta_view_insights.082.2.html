Downloading New and Updated Web Page Insights
To download and install the latest insights for Web Transaction Analysis, do the following:
1)
In the Desktop Console, choose Insights > Update Center.
2)
Navigate to the following section in the insights table:
support.opnet.com/ace_live/insights/modules >
      Page Analysis Insights
If you do not see this in the table, click Manage Servers and add the following server to the server list:
support.opnet.com/ace_live/insights
