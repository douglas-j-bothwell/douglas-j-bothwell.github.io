AS Topology
The AS Topology window allows you to examine autonomous system connectivity to a destination. It shows the service providers that data packets must transit in order to reach a remote autonomous system.
The data displayed in AS Topology is collected from traceroute. The AS path from the local network to the destination IP address is determined by mapping each hop in the traceroute to an AS number using the WHOIS Database. A Destination AS cannot be displayed in the AS Topology view if a traceroute to that destination is not stored in the database. To minimize this likelihood, the automated traceroute selection algorithm chooses traceroutes across active Dest AS groups (described in Automated and Manual Traceroutes). Configure a preferred IP address within a Dest AS to ensure that a traceroute is always performed to a destination within the AS (see Preferred IPs).
To invoke the AS Topology views, select View > Topology > AS Topology.

AS Topology Window
The AS Topology window is blank by default. There are two ways to display the AS connectivity to a destination:
Drag and drop a Dest AS group from another console window (such as a table) into the AS Topology window.
Select an AS from the Traced AS list and click the Go button. This is a list of all AS for which there is a traceroute stored in the appliance database. If an AS does not appear in this list, then it cannot be displayed in the AS topology window.
To select multiple ASes, click ctrl+click or shift+click.
Note: Only Dest ASes with traceroutes stored in the database are displayed in the chart pane.
The AS topology window shows the AS connectivity between the local network where the appliance is installed and the remote AS groups. The index lists the ASes that are currently displayed. The AS path for each AS can be viewed by double-clicking on the AS group in the index.

AS Topology Toolbar
The following figure shows the options available on the AS Topology toolbar.
Figure 20
AS Topology Toolbar
 
The list below describes the functions of the controls on the top from left to right:
Clear Graph - Clears the current graph from the display pane.
Selection Mode - Select a node in the graph. The selected node is subject to the next operation.
Zoom mode - Click and sweep on areas to zoom in on.
Fit Content - Adjusts the zoom level so that the graph fits inside the display pane.
Zoom To 100% - Shows the graph to scale.
Zoom In x2 - Doubles the zoom level.
Zoom Out x2 - Halves the zoom level.
Select Background Image - Sets background image in the display pane. The image can be removed by right-clicking in an empty part of the display pane and selecting Clear Background from the menu.
Delete - Deletes selected groups from the graph.
Export Graph - Exports the current graph to a .gif, .jpg or .png file.
Help Pages - Shows the Operations Manual help pages.
Layout - Select the layout view of the graph in the display pane. There are five to choose from: Tree (Horizontal), Tree (Vertical), Organic, Circular and Orthogonal.
