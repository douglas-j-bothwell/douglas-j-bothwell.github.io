Report Templates
Report Templates enable you to define advanced reports quickly and easily. Unlike most other reports, which consist of a single table or chart, reports based on templates can include multiple elements to provide a more comprehensive report similar to an insight.
A report template has the following configurable attributes. The set of available attributes depends on the specific template.
Selected Business Groups / Applications / metrics
Number of results (topping)
Report/chart/table titles
All report templates, and all reports based on these templates, are available in the Report Manager. On 8.5.3 and higher appliances that have been updated from a previous release, legacy reports (those defined on the appliance before the update) are available on the Insights > Reports menu.
To define a new report from a report template, do the following.
1)
Open the Report Manager (Desktop Console > Report Manager).
2)
Click New. The Edit Report Content dialog box appears.
3)
In the "Please select a report content" menu, select Report Template. The Default Report Templates dialog box appears.
Figure 1
Selecting the “Report Template” Type
 
4)
Browse through the report templates until you find the template that matches most closely the report you want to create. Note that you can see a preview and description of each template by selecting it in the table.
Figure 2
Default Report Templates
 
5)
Select the template and click OK. The Select/Specify Settings dialog box appears. (An Edit Report Content dialog also appears behind the Select/Specify Settings dialog box.)
Every report template has its own set of default settings for the report content: Business Groups, Applications, Metrics, Number of Results. The available settings in the Report Content page depend on the specific template you are configuring.
6)
Edit the settings in the Report Content page to specify the content you want to include in your report.
7)
Click the Report/Chart Title tab.
8)
Edit the titles for the report and the individual charts to describe the content in your report.
Note—You must specify the names of the Business Groups, Applications, and Metrics included in your report. If the default title includes a string such as "[Business Group]" or "[Application]", you must specify the literal string (such as "Default-192.168.x.x" or "HTTPS").
9)
Click OK. The Select/Specify Settings dialog box closes and the Edit Report Content shows a preview of the report content.
10)
Review the report content and verify that it shows the information you want. Click OK to proceed. Otherwise, click Cancel and start from the beginning.
11)
In the Report Manager, edit the other settings for the report.
