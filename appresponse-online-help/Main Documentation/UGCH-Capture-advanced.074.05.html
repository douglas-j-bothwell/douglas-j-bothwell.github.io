High Speed Capture Dashboard
You can use the High Speed Capture Dashboard (Insights > Diagnostics > High Speed Capture Dashboard) to monitor the status of High Speed Captures on your appliance.
Figure 3
High Speed Capture Dashboard
 
This insight shows
Rolling Buffer time range and size (in bytes)
Snapshot Buffer time range, size, and number of alerts
Metrics for individual monitoring interfaces and all interfaces
–
Throughput (Average/Max)—Throughput observed on each monitoring interface and all interfaces
–
Disk Throughput (Average/Max)—Rate at which packet data is written to disks on the appliance and any attached Expansion Chassis
–
Packet Size (Average/Max)
–
Packet Drops (Average # of dropped packets per second)
Time series chart showing Average Throughput and Disk Average Throughput for the current project time window
Note: This insight is supported on ACE Live 8.0.8 (or higher) and AppResponse 8.5.2 (or higher) only.
Note: This insight shows data for the current time window. If the project time is longer than 24 hours, this insight might take several minutes to update with all the packet capture data.
