Reports in Director Domains
The director can access any data from any appliance in a domain transparently. As a result, in domain environments, it is not necessary to configure reports on individual appliances. Rather, reports can be executed and examined centrally on the domain director.
Sharing Report Definitions
The Report Manager allows you to export and import report definitions in XML format to facilitate sharing reports between appliances.
To export one or more report definitions, open the Report Manager, select the desired reports for export, then select File > Export from the main menu to display the Export Report File dialog box.
Figure 1
Report Manager - Export Report File Dialog Box
 
You may export selected reports only, or all the reports in the Report Manger.
To import reports, open the Report Manager and select File > Import from the main menu to bring up the Import Report File dialog, which operates in similar fashion to the export dialog illustrated above.
