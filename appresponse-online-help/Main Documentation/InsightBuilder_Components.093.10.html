View Component
There is only one View Component available for building Insights: the Alert Log Component.
Alert Log
The Alert Log Component is used to display Static, Static-Any, and Adaptive Alerts, and to send Interactions containing information from either the Alert event (the actual triggered alert) or the Alert task (the definition of the Alert) to other Components.
Edit the Alert Log Component
The Alert Log Components is edited using the Alert Log Controls Dialog or by directly setting controls.
Interactions Produced
The Alert Log Component produces the following Interactions.
 
Table 58
Interactions Produced by Alert Log Component 
Interaction
How Produced
group
User selects an Alert in the Alert Log. The traffic group that corresponds to that Alert is sent in the Interaction.
alert
User selects an Alert in the Alert Log. The task that corresponds to that Alert is sent in the Interaction.
The Alert Log Component does not send out any Interactions when it is created.
