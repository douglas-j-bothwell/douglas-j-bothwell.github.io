SLA Dashboards
Service Level Agreement (SLA) Dashboard enables users to see, in a quick glance, the service levels of specific applications and business groups. You can specify SLAs based on up to three metric thresholds (combined by AND / OR booleans). You can use Web Console SLA dashboards to view “real-time” SLA compliance for up to the most recent 15 minutes, hour, or day. (SLA dashboards in the Desktop Console are more “historical” in that they report on past days, weeks, and months.) Web Console SLA dashboards also support Web Applications in addition to Defined and Other Applications.

Important Notes
Note the following:
You cannot include more than 500 elements (Business Groups x Applications) in one SLA Dashboard.
Web Console SLA Dashboards are available on appliances only, not directors.
If an SLA has multiple metric conditions, the Root Cause Analysis shows a separate Time Series Chart for each metric.
The Web Console supports the configuration of real-time SLA Dashboards only. Go to the Desktop Console to configure historical SLA Dashboards. You can view both types of dashboard in the Web Console.
Create a New SLA Dashboard
Procedure 1
To create a new SLA Dashboard:
1.
Make sure that you are logged in to the appliance as a user with Administrator or Standard access privileges.
2.
Home tab: Select Administration, and then “SLA Dashboards” from the category table listing.
3.
SLA Configuration tab: Click New.
4.
Edit: New SLA Dashboard tab: Specify the content and layout options you want. Descriptions are here:
• 
Table 1 SLA Dashboard Options: Content Selection & Layout
• 
Table 2 SLA Dashboard Options: Default Analysis Settings
Note: Click the “default” label to override the default thresholds for a column, row, or individual element in the Dashboard. (You must be in Edit mode to do this.)
 
5.
At the top of the page, choose File > Properties and specify the following:
• 
Name—Name of the dashboard as it appears in the tab and the SLA Dashboards treeview in the Home tab.
• 
Description— This description appears when you click on the parent category in the Home tab > SLA Dashboard treeview.
• 
Category—The parent category of the SLA Dashboard that appears in the Home tab > SLA Dashboard treeview.
6.
Choose File > Save to save the new custom view.
7.
Go to the Home tab. In the treeview, select the SLA Dashboard category you assigned. Verify that the dashboard appears in the SLA Dashboards table.
End of Procedure 1
