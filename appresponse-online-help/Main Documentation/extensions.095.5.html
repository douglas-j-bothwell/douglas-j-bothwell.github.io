ScriptData: Passing data from the Insight Extension to the Insight
The ScriptData API allows users to pass information from an Insight Extension back to an Insight. The information may be translated back into an Insight Interaction by the component that is running the Insight Extension or it may be handled in a different manner. The script passes information back to the insight by echoing it to the output stream. The data is always preceded by the ScriptData tag. The script data tag is followed by a key and then the options for that key as follows:
      ScriptData:Key:Options
The following keys are available:
ScriptData:BOXES:[boxName1,...,boxNameN]—specifies the names of the boxes to be passed out as a box Interaction.
ScriptData:IP:[ip1,...,ipN]—specifies the IPs to be passed out as a group Interaction
ScriptData:AS:[as1,...,asN]—specifies the AS numbers to be passed out as a group Interaction
ScriptData:GROUP:[groupPath1,...,groupPathN]—specifies the group paths to be passed out as a group Interaction
ScriptData:METRICS:[metric1,...,metricN]—specifies the metric names to be passed out as a metric Interaction
ScriptData:INTERVAL:[start,end]—specifies the interval to be passed out as an interval Interaction
ScriptData:LIMIT:[integer]—specifies the integer value to be passed out as a value Interaction
ScriptData:LAUNCH:[insight name]—Launches the insight with the specified name.
ScriptData:TIMEDATA:[XValues:comma separated list of raw x-data;YValues:comma separated list of raw y-data;XUnit:x-units in the UI units format;YUnit:y-units in the UI units format;Color:the rgb value for the display color of the data set as r:g:b;Style:the style to use in rendering the data set (Line, Point);Label:the text to display as the label for the data set- passes time series data to the chart]
ScriptData:ANALYSISRESULT:[AnalysisName:a String with the analysis name;AnalysisMessage:a String with the analysis message;Group:the group the analysis was done on;Metrics:the metrics that were used in the analysis;Issues:a comma separated list of issues of the form start:end:severity]
ScriptData:ERROR:[string]—Launches an error dialog that displays the string as an error message.
ScriptData:DEBUG:[true, false]—if true, prints an error messages written to standard error to the console window.
ScriptData:SCROLL:[start,end]—scrolls to the start or end of the document.
ScriptData:INDEX:[integer]—specifies the index to be passed out as an index Interaction
ScriptData:STEP:[stepName]—specifies the name of the layer to switch to.
ScriptData:TEXT:[string]—specifies a String to be passed as a text Interaction.
When running an external script from an Insight, you can pass multiple instances of the ScriptData:TEXT option back to the Insight. For example, you can pass several strings from one run and then use these to populate multiple text boxes / labels in the Insight. To pass multiple strings, add an “index” to each instance of ScriptData:TEXT. For example:
ScriptData:1:TEXT:This is test 1 text.
ScriptData:2:TEXT:This is completely different text.
When you create the text interaction, you can right-click on the link to change the “Producer” option to reference these different indexes.
ScriptData:INPUT:[type1,label1,key1,initValue1;…;typeN,labelN,keyN,initValueN]
The following example shows the different ways that Insight Extensions and Scripts can pass data back to an Insight:
#!/bin/sh
# This script demonstrates the API for returning data from a script to an insight.
# The following parameters should be passed to this script:
# parameter 1 - Group Path
# parameter 2 - Group Name
# parameter 3 - Interval
# parameter 4 - Metrics # parameter 5 - The start of the interval # PATH=/bin:/usr/sbin:/usr/bin:.
 
start=$((${5}+300))
end=$((${start}+3600))
echo "\"
echo "<html><H2>Retrieving Data ...</H2></html>"
sleep 5
echo "ScriptData:BOXES:cuda1"
echo "ScriptData:IP:10.10.0.162,10.10.0.255"
echo "ScriptData:METRICS:TPIO,TPO,TPI"
echo "ScriptData:INTERVAL:${start},${end}"
echo "ScriptData:LIMIT:2"
echo "ScriptData:TEXT:The Groups and Metrics were passed in from the Script"
echo "ScriptData:GROUP:v4.0.0:appGroup 0/0/Internet,v4.0.0:appGroup 0/0/Internet - CNN"
echo "\"
echo "<html>" echo " <H2>Data Successfully Passed to Insight</H2>" echo "</html>"
