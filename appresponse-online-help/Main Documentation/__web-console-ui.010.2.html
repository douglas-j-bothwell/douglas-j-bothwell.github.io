Time Controls
Every tabbed page includes its own separate time controls and toolbar. Unlike the Desktop Console, the Web Console does not have a universal Project Time that applies to most views. All time toolbars reflect the time zone on the local host—that is, the local time where you are viewing the console—not the time zone where the appliance is located.
