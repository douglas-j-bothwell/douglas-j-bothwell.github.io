Update Center
The Update Center (Desktop Console > Insights > Update Center) enables you to install new and updated insights. It is good practice check the Update Center periodically for updates to currently installed insights.
The following topics are discussed:
Managing Update Servers
Installing New/Updated Insights
Automatic Update Checking
Insights-Only User Account
Updating Insights in Secure Environments
Managing Update Servers
To add or remove servers from the list that is used for Insight updates, click the Manage Servers button in the Update Center.
Figure 1
Manage Update Servers Dialog Box
 
Select the Visible check box to make a server appear in the Update Center. Select the Check for Updates check box to have the Desktop Console automatically check the server for new or updated Insights.
Adding a New Update Server
Your reseller, integrator, or IT department might maintain an update server in addition to those provided by Riverbed. To add such a server to the Update Center, click Add, enter the URL of the update server in the Connect to Update Server dialog box, and click OK. The new server appears in the Manage Update Servers dialog box and you can configure it for use.
Use the Remove button to remove Insights added in this way, or click Defaults to return to the standard selection of Riverbed update servers.
Installing New/Updated Insights
The left-hand pane displays all the new and updated Insights available to you. Please note that you only see the Insights that are compatible with your console. Click any Insight to see a more detailed description of it in the right-hand pane. Select the Insights you want to add or update, then click Install to download and install them. A progress bar appears at the bottom of the description pane; click Stop to cancel the download process.
Automatic Update Checking
When you connect to an appliance, the Desktop Console automatically checks specified servers and notifies you if updates are available. You can specify which servers are checked in the Manage Update Servers dialog box.
Figure 2
Insight Updates Available
 
Click OK to open the Update Center and download the new or updated Insights.
Insights-Only User Account
A special user account is available that can only access Insights from the Desktop Console. It is assigned using the User Admin Manager (see User Admin Manager).
Updating Insights in Secure Environments
You can install new insights and software patches on secure appliances (that is, appliances deployed in secure environments that have no internet access).
Note the following:
You can use this workflow to install insights and software patches, but not full updates, on a secure appliance.
This workflow requires that you have at least one AppResponse appliance that is not in the secure environment. If you do not have any appliances outside of the secure environment, contact Riverbed Technical Support.
To install one or more insights or software patches on a secure appliance, do the following:
1)
On the unsecured appliance, download and save the update fil(s) to a recordable medium such as a USB memory stick or CD/DVD-ROM:
a)
Open a Desktop Console and connect to the appliance that is outside the secure environment.
b)
Choose Tools > Preferences. In the Insights tab, verify that the Enable Insight Download from Update Center option is selected.
c)
Choose Insights > Update Center.
d)
Select the Update Server and then the updates you want to install.
e)
Click Download. AppResponse stores the updates in your Local Insight Directory. (The Local Insight Directory path is specified in the Desktop Console > Tools menu > Preferences dialog box > Insights tab.)
f)
Save the contents of the Local Insight Directory to the USB stick or other medium.
2)
Install the update file(s) from the recordable medium onto the secure appliance:
a)
Bring the recordable medium into the secure environment and copy the contents into the Local Insight Directory specified for the secure appliance. (If you do not know the location of this directory, log in to the secure appliance, open a console window, and check the Tools menu > Preferences dialog box > Insights tab.)
b)
Log in to the secure appliance and open a console window, if you have not already done so.
c)
Choose Insights > Update Center.
d)
In the Update Center dialog box, click Edit Servers.
e)
In the Edit Servers dialog box, select the Visible checkbox for Local Insight Directory and click OK.
f)
Select the Local Insight Directory and the updates you copied into this directory. Then click Install. The local updates are installed on the secure appliance.
