Working with Analysis Components
This section briefly reviews how to add, configure, rename and remove Analysis Components. For more detailed information on specific Analysis Components.
With a few exceptions, Layer-related Analysis Component operations are discussed in the section. See Working with Layers.
Adding an Analysis Component
To add an Analysis Component, first select the desired Layer in the Element pane, then select the desired type of Component in the Analysis Component drop-down pick list in the Insight Development Kit toolbar area, then click Add Analysis.
Figure 28
Add an Analysis Component
 
This brings up the Set Analysis Parameters dialog for the selected Analysis. Instructions for setting the parameters of an Analysis Component can be found in the section for each Analysis Component in Analysis Components.
Click OK on this dialog to add the Analysis Component.
Configuring an Analysis Component
To configure an Analysis Component, select it in the Element pane, right-click to display the Element Pane context menu, and select Configure Analysis to display the Set Analysis Parameters dialog. Instructions for setting the parameters of an Analysis Component can be found in the section for each Analysis Component in Analysis Components.
Click OK on this dialog to add the Analysis Component.
Removing an Analysis Component
To remove an Analysis Component, select it in the Element pane, right-click to display the Element Pane context menu, and select either Remove to remove it from that Layer only, or Remove from All Layers to remove it from all Layers in the Insight. You can also select the Component node in the Workflow View, right-click it and select Remove Component.
Renaming an Analysis Component
Renaming an Analysis Component is the same process as renaming a Visual Component (see Rename Visual Components): select it in the Element pane, right-click to display the Element Pane context menu, and select Rename to display the Rename dialog, or select the Component node in the Workflow View and right-click for the context menu.
