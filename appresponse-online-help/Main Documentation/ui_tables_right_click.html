Table Right-Click Menu Operations
Both Group and Favorites tables offer an extensive context menu invoked by right-clicking on a cell, row, column, or anywhere in the table. The Favorites table offers some context menu choices not available with a Group table. The context menu choices are listed below in alphabetical order. They are context-sensitive to the table that you have selected, so not all menu items are available at any given time.
 
Table 7
Table Context Menu Definitions 
Menu Item
Definition
Define
Define an Alert based on the right-clicked row (group) and column (metric).
Duplication Table
Duplicate the current table. The new table is automatically assigned a unique name.
Edit Columns...
Add/delete metric columns.
Email Snapshot...
Email a snapshot of the current table.
Snapshots are emailed from the appliance or director to which the project is connected, not the computer hosting the Desktop Console.
Expand
Expand the selected drill-down.
Hide/Show Toolbar
Toggle between showing and not showing the tool bar on the selected table.
Import Groups...
Import group files - Favorites table only.
Insert Column
Add/Delete metric columns to the selected table.
Mark to Export (On/Off)
Mark the selected cells for export to a CSV file.
Network Tools
Launch a third-party network tool. For information about integrating third-party tools with AppResponse, see Integrating with External Applications.
Properties
Select and right-click a Business Group to open the Business Group Manager to see the properties.
Publish
Publish a Report or a Snapshot of the table.
Remove Column
Remove metric column from selected tables.
Selection Mode
Specifies the selection mode for Shift-click and Ctrl-click operations when selecting and copying/pasting content from tables:
•
Multiple Rows—Select one or more rows using click, Shift-click, and Ctrl-click
•
Multiple Cells—Select one or more rows using click, Shift-click, and Ctrl-click
•
Single Row—Click in a row to select it (Shift-click and Ctrl-click disabled)
•
Single Cell—Click in an individual cell to selected it (Shift-click and Ctrl-click disabled)
The setting of Selection Mode is ignored for context menu operations (such as Mark for Export), when dragging and dropping the selection into another window, i.e,. you cannot use it to make selections for such operations.
Send To
Send the selected data to a Chart or Favorites Table.
Set Column Units
Set the units shown in the metric columns.
Show Group Path
Select and right-click group to select if show by Full, First Element or Last Element in the path.
