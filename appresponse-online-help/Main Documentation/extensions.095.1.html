Insight Extensions and Scripts
This section covers Insight Extensions and Scripts. Remote scripts that are located on the appliance are called Insight Extensions. Insights have the ability to run two types of scripts:
Remote Scripts (Insight Extensions) that are stored on the appliance and written in any language that is supported on the appliance (unix shells, perl, java).
Local Scripts that are stored on the user's hard drive and written in the native platform's language (dos, unix shells).
