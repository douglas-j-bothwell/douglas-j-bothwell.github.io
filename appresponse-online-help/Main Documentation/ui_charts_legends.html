Chart Legend
The chart legend provides a color index which allows you to identify a group or metric present in the chart.
Legend Right-Click Menu Operations
The chart legend includes a context-sensitive menu for modifying the chart content. Right-click in the legend and select one of the following operations.
 
Table 1
Chart Legend Right-Click Menu Operations
Operation
Description
Settings
Invoke the Chart Settings dialog box.
Set Color
The color assigned to a group can be changed through the chart legend. Right-click a group within the legend and select Set Color to invoke the color selection dialog.
Remove
Remove the selected group from the chart.
Remove All
Remove all groups from the chart.
Hide Legend
The chart legend can be hidden by right-clicking on the legend and selecting Hide Legend. It can be returned to normal size by double-clicking on the chart to pull up the Chart Settings dialog. Select the Legend tab and then Show Legend.
Place Legend in Corner
Place Legend at Bottom
By default, the chart legend appears directly below the chart. The Place Legend in Corner selection places the legend in the upper right corder of the chart. When the legend is in this location, the context menu provides the Place Legend at Bottom option which returns the chart to the original location.
Send to > New Favorites Table
You can send the traffic for a Business Group to a new Favorites table. Right-click on the group name in the chart legend and then choose Send To > New Favorites Table.
Show Group Path
By default, the legend displays only the end node of the group path (for more information on group path, see Drill-Downs and Metric Availability). Use the Show Group Path selection to toggle display of
•
The complete group path (example: Total Traffic > TCP > Applications > FTP). This option is always recommended for metric drill-downs, because the same child path element can have multiple parents. When the full group path is displayed, the context menu provides the Show Short Group Path option to return group display to the original behavior.
•
The first path element (example: Total Traffic)
•
The last path element (example: FTP)
In many cases, the abbreviated path for a group is the same as the full group path. This is the case if an entire group (e.g., Business Group) is charted rather than a drill-down.
Show Information
Show Group Name
The Show Information option causes the chart to display the information field for the group, rather than the Name field. For instance, when an IP address is displayed in the chart, the chart displays the hostname for that IP address resolved through DNS if the Show Information option is enabled. Select Show Group Name to return to the default behavior.
Set as Project Time
Sets the Project Time to the current X-Axis zoom level. All charts and tables in the project then display the same time selection you zoomed to in the chart—in effect, the whole project has zoomed in on whatever anomaly you want to inspect.
When you click Set as Project Time, a Reset Auto Zoom Project Time button appears in the project toolbar. Clicking this button to restore the original time selection.
Duplicate Chart
Create an exact duplicate of the selected chart. This option is useful when you have a multi-group time series chart displaying a set of groups and you’d like to examine a different metric for these groups alongside the original chart.
Publish
•
Publish Report—Create a report from the chart (as described in Report Manager)
•
Publish Snapshot —Create and publish a snapshot of the chart (as described in Snapshots).
Email Snapshot
Email a snapshot of the current chart from the appliance or director of the current project (i.e., not from your local host).
For more information about email settings, see Configuring Outgoing Email Parameters.
