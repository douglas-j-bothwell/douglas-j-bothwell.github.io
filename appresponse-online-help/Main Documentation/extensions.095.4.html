Variables: Passing Data from an Insight to an Insight Extension
Information can be passed to scripts using variables that output information that has been passed to the Insight Component that is running the script via an interaction.
The following variables are available:
ip—inserts text with the IP address for any IP groups passed in a group Interaction.
ip1—inserts text with the first IP address in a drill-down for any groups passed in a group Interaction.
ip2—inserts text with the second IP address in a drill-down for any groups passed in a group Interaction.
as—inserts text with the as number for any AS groups passed in a group Interaction.
group—inserts text with the name of the last group in a drill-down for any groups passed in a group Interaction.
groupName—inserts text with the name of the last group in a drill-down for any groups passed in a group Interaction.
groupFirstName—inserts text with the name of the first group in a drill-down for any groups passed in a group Interaction.
groupLastName—inserts text with the name of the last group in a drill-down for any groups passed in a group Interaction.
groupFullName—inserts text with the full group names for any groups passed in a group Interaction.
groupDesc—inserts text with the group descriptions for any groups passed in a group Interaction.
groupPath—inserts text with the official group path strings for any groups passed in a group Interaction.
interval—inserts text with the interval passed by an interval Interaction.
start—inserts text with the start time (in seconds) passed by an interval Interaction.
end—inserts text with the end time (in seconds) passed by an interval Interaction.
timezone—inserts text with the timezone offset from GMT.
metrics—inserts text with the metric names passed by a metric Interaction.
boxes—inserts text with the box names passed by a box Interaction.
boxIds—inserts text with the box ids passed by a box Interaction.
tasks—inserts text with the task names passed by a task Interaction.
limit—inserts text with the limit passed by a value Interaction.
text—inserts the text passed by a text Interaction.
The following Insight Extension is executed in this example:
#!/bin/sh
# This script demonstrates the API for passing data from an Insight to a Script.
# The following parameters should be passed to this script:
# parameter 1 - Group Path
# parameter 2 - Group Name
# parameter 3 - Interval
# parameter 4 - Metrics
# parameter 5 - The start of the interval
# PATH=/bin:/usr/sbin:/usr/bin:.
echo "\"
echo "<html>"
echo " <H2>Insight Variable Parameters:</H2>"
echo " <ul>"
echo " <li>  Group Path: $1</li>"
echo " <li>  Group Name: $2</li>"
echo " <li>  Interval: $3</li>"
echo " <li>  Metrics: $4</li>"
echo " <li>  Start: $5</li>"
echo " </ul>"
echo "</html>"
