Database Statement Troubleshooting Insight
This insight is useful for troubleshooting a specific SQL statement by viewing data for all transactions that use the SQL statement of interest.
Note: This insight is not available from the Insights menu. To access this insight, go to the Database Troubleshooting Insight, select a statement in the Slowest Transactions tab, and click Troubleshoot SQL Statement Used in Selected Transaction.
Figure 4 Database Statement Troubleshooting Insight
 
To troubleshoot the SQL statement of interest, you can
View the database response times and transaction count for all queries that use the SQL statement of interest (Response Time and Transaction Count time charts).
You can also zoom in to a time window of interest by dragging horizontally in a time chart.
Download packets for a transaction: right-click on the transaction and choose Download Packets.
Break down the total response time or transaction count by database, user, application, and so on (Response Time Breakdown and Transaction Count Breakdown charts).
View metrics for the slowest SQL queries (Slowest Transactions table).
