Installing the Database Performance Module
The Database Performance Module is installed automatically on AppResponse Appliances along with the AppResponse software (release 8.5.2 RC3 or later). However, it is disabled initially. To enable it, perform these steps:
1)
License the Database Performance Module.
2)
Update to the Latest SteelCentral AppSQL Patch, if necessary.
3)
Update to the latest AppSQL Insights, if any.
Instructions are given in Database Insights.
4)
Define Monitoring Rules.
License the Database Performance Module
Use the following procedure to enter the license information supplied by Riverbed.
Procedure 1 Licensing the Database Performance Module
1.
Open the Management Console (as described in Access the Management Console).
2.
Click Manage Licenses.
3.
Click Select.
4.
Click New License.
5.
For each license information field, enter the value provided by Riverbed.
Note—License information is case sensitive. You must enter all information exactly as provided by Riverbed.
6.
After entering all license data, click Save.
7.
Click Done to close the “Add new license” page.
8.
Click Done to close the “Manage licenses” page.
End of Procedure 1
 
 
Update to the Latest SteelCentral AppSQL Patch
Use this procedure to update the software on your Appliance with the latest SteelCentral AppSQL patch.
Procedure 2 Updating the SteelCentral AppSQL Software
1.
In AppResponse, open a Desktop Console.
2.
Choose Insights > Update Center.
3.
Patches are available from the following server:
support.opnet.com/ace_live/insights/support/
If this server is not listed in the Update Center, click the Manage Servers button and add it.
4.
Check the Patches section for an SteelCentral AppSQL patch. If there is one, select it and click Install.
End of Procedure 2
