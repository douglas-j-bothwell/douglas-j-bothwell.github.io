Important Notes
Note the following:
These Insights are available only when a Database Performance Module license is installed and SQL data collection is enabled, as described in Installing the Database Performance Module.
To install the latest Database Insights from the AppResponse Desktop Console, follow Procedure 1 Installing the Database Insights.
You must install these insights on the AppResponse appliance itself; you cannot run them locally.
Most of these insights include a DNS option in the top right corner. When this is set to On, the insight shows DNS names instead of IP addresses for database servers and clients. Enabling this option can slow down insight performance because the appliance needs to perform periodic DNS queries.
The Database Performance Insight and Database Troubleshooting Insight include a Transaction Rate time series graph that includes an Average Indeterminate Transaction Rate (#/min) metric. Database Performance Module marks a transaction as “indeterminate” when it is not known whether the DB transaction ended successfully. The most common cause for this is packet loss, typically due to SPAN port oversubscription, where the packets which would have indicated the completion status were missing. These insights ignore all indeterminate transactions when they calculate database metrics.
Indeterminate transactions usually indicate that the appliance is not observing all packets for these transactions. If the Transaction Rate graph shows indeterminate transaction rates higher than 0/min, you might want to investigate possible causes such as
–
One or more SPAN ports are oversubscribed.
–
The connection between the appliance and a SPAN port is faulty.
–
The appliance or database host was recently rebooted.
–
Some network devices use unidirectional routing.
Queries might take several seconds to collect all requested data. For this reason, these Insights provide a data query progress bar and a Stop Query button in the bottom-right corner.
 
 
Procedure 1 Installing the Database Insights
1.
Open an AppResponse Desktop Console.
2.
Choose Insights > Update Center.
3.
Insights are available from the following server:
support.opnet.com/ace_live/insights/modules/
If this server is not listed in the Update Center, click the Manage Servers button and add it.
4.
Check the Database Insights section for available Insights. If there are any, select them and click Install.
End of Procedure 1
