These metrics are available for viewing within tables in the various Database Performance Insights. Most of the metrics are hidden by default. To add them to a table, follow the procedure in Displaying Metrics in an Insight.

Displaying Metrics in an Insight
Most metrics are hidden by default. Use this procedure to add hidden metrics to a table or remove visible ones.
Procedure 5 Adding Metrics to a Table
1.
Right-click in one of these Insight tables:
 
Database Browsing Insight Statements table (right)
 
Database Performance Insight DB Servers table (bottom left)
 
Database Troubleshooting Insight DB Instances or DB Applications table (bottom left)
 
Database Statement Troubleshooting Insight Slowest Transactions table (bottom)
2.
Choose Show Toolbar from the pop-up menu.
3.
Click the Edit Columns icon () in the table toolbar.
4.
Select the desired metrics.
5.
Click OK.
End of Procedure 5
 
 
Displaying Port Metrics
Values for the DB Port and Server Port metrics are not captured by default. To make them available, follow this procedure.
Procedure 6 Capturing Port Metrics
1.
Open the Management Console (as described in Access the Management Console).
2.
Click Manage Software Instances.
3.
Click Manage Instance.
4.
Click Advanced Configuration.
5.
Click on the active configuration.
6.
Click Dispatcher.
7.
Select the checkbox for the append_ports parameter and set the parameter value to true.
8.
Click Save changes.
9.
Click Save as active config.
10.
Click Stop, then click Start.
End of Procedure 6
