Channel Metrics
The appliance collects metrics for the media streams associated with each observed call. A channel can be either Video or Voice. You can view the Channel metrics for each call in the VoIP web UI > VoIP Data > Call Lookup page.
Metrics are computed for each minute of the call and the data point for each minute is representative of all the calls that were active at some point during the minute. This provides a real-time view of call quality, call volumes, and other call data.
