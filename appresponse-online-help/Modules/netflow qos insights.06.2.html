Applications for Selected ToS
This Insight is useful for identifying the applications with a specified ToS (DSCP) value that are seen on an interface. This allows you to verify that the expected applications are observed. You can also identify unexpected applications (for example, re-marking errors) and discover which servers and clients were affected.
You can open this Insight from the Insights > QoS submenu.
Figure 1 Applications for Selected ToS Insight
 
The following procedure describes the suggested workflow for this Insight.
Procedure 1 Applications for Selected ToS: Suggested Workflow
1.
Locate and select an interface in the Interface Selector table (top left).
A breakdown of ToS traffic for the selected interface appears in the Breakdown chart (top right). Each ToS is identified by a DSCP value.
2.
Select the chart view you prefer (summary or time history).
3.
In the Breakdown chart, select a ToS by clicking on either a legend or a data area.
The legend and data area are highlighted in the chart and the Top Apps table (bottom left) is populated with the top applications for the selected DSCP.
4.
Select an application in the Top Apps table.
The Top IPs table (bottom right) is populated with the leading IPs using that application.
5.
If desired, you can expand an entry in the Top IPs table to see what other IPs it communicates with using the selected interface, DSCP, and application.
End of Procedure 1
