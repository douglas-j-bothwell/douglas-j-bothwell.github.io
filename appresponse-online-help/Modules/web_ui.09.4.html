VoIP Data Settings
The VoIP Data Settings section has the following pages:
Call Quality Thresholds Pages
VoIP Call Metrics Page
Failed Call Configuration
Time Zone Page
Call Quality Thresholds Page
This page enables you to edit the default MOS thresholds used to define Better, Fair, Poor, and Worst call quality for Call Metrics.
VoIP Call Metrics Page
The VoIP Call Metrics page (VoIP Data Settings > VoIP Call Metrics) lets you specify which metrics should be included in the call list that is displayed after a call lookup.
Failed Call Configuration
The Failed Call Configuration page lets you specify the definition of a failed call and is used for the Failed Calls metric in the Desktop Console.
Time Zone Page
The Time Zone page allows you to view the information in an appropriate time zone for you. The system supports using the appliance time zone as a frame of reference or that of your local client accessing the system. A setting of GMT is also provided.
Once selected, information is searched and displayed relative to your selection. This option can be changed at any time.
To go to the Time Zone page, choose VoIP Data Settings > Time Zone in the Module for VoIP Performance menu.
