Auditing Actions and Users of Database Performance Insights
The Database Performance insights generate log messages to track user actions. This log shows the following information about each action performed using any Database Performance insight:
Query time
Insight used
Action performed
User information: login name, client IP, and access privileges on appliance
Database Performance logging requires the following versions of these insights:
Database Browsing--1.1.1 or higher
Database Performance--1.1.1 or higher
Database Troubleshooting--1.1.2 or higher
You can view log data in the Desktop Console > Insights > Database > Database Audit Log Viewer.
You can also view log data using an external SSH client on a remote host.
To view a list of available logs, enter:
ssh [user]@[appliance] cliext ASXAuditLogViewer listLogFiles
To download contents of an individual log to a file on the UNIX host, enter:
ssh [user]@[appliance] cliext ASXAuditLogViewer viewLogFile [logname] > mylog.txt
