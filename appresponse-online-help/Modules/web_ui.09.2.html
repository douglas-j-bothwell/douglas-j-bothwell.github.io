VoIP Data
You can view VoIP data on the following pages in the Web UI:
Call Lookup Page
Alert Lookup Page
These pages can be reached from links in the VoIP Data section of the Module for VoIP Performance menu.
Call Lookup Page
The Call Lookup page lets you display information about a call or group of calls. You also can view detailed information about a specific call or its channels. To view call information, follow this workflow:
1)
Go to the Call Lookup page (choose VoIP Data > Call Lookup in the Module for VoIP Performance menu).
2)
Display a Call List:
–
Click Call ID to display calls by ID, or
–
Click Search to display calls for a specific time period, optionally filtered by various call metrics
3)
View details about all calls in the displayed group or about a specific call or call channel:
–
Call Info—detailed information on a single call
–
Signaling—call ladder for a single call
–
Statistic Charts—summary information about all displayed calls
–
Topology—connection diagrams for all calls
Call ID
The Call ID link on the Call Lookup page lets you look up calls by call ID. To display a list of calls, enter a comma-separated list of one or more call IDs, then click Go.
Search
The Search link on the Call Lookup page lets you look up calls by searching on a time period and some combination of call metrics. The search criteria for looking up calls are organized into two groups: general and advanced.
General Search Criteria
The search criteria on the General link are:
Time—Select calls for a specific time period. (This criterion is required for all searches.)
Endpoints—Select calls based on the IP address or business group of the originator or receiver. You can specify IP addresses using standard CIDR (Classless Inter Domain Routing) address notation.
Phone number and name—Select calls based on the name or phone number of the source, destination, or both. You can enter a complete phone number or a partial one using the standard wildcards ‘*’ (any characters) and ‘?’ (any single character).
Signaling Type—Select calls based on signalling type. Select the checkbox for each signalling type to be included in the search.

Advanced Search Criteria
The search criteria on the Advanced link are:
Quality—Select calls based on MOS CQ, network loss rate, jitter (PPDV), degradation, and codec.
Setup—Select calls based on setup time and error code.
Duration—Select calls based on call duration.
Channel count—Select calls based on the number of channels.
Call Type—You can now choose to search for Voice calls, Video calls, all calls, or calls whose type is unknown.
Conference calls—The “Show only conference calls” checkbox enables you to filter out non-conference calls from the search results.
Other—Select calls based on media type and precedence/DSCP.
For media type, enter a comma-separated list of values. You can include any value your VoIP infrastructure uses, such as “audio”.
For ToS IP Precedence/DSCP, enter a comma-separated list of values and use the drop-down menu to specify whether calls should match or not match these values. Allowed values for precedence/DSCP are
–
Integer value in the range 0–63; interpreted as DSCP (first 6 bits of the ToS byte)
–
Integer value in the range 0–7, prefaced with “tos”; interpreted as ToS IP Precedence (first 3 bits of the ToS byte)
–
DSCP mnemonics, such as EF and AF10
Any of the precedence values can be expressed as a range by using a “-” (dash). For example, a valid string for the precedence/DSCP field is
“EF,44, tos 3-4,5,47-52,AF11,tos 2”
Note: All criteria entered on the General tab carry over to the Advanced tab.

Looking Up Calls by Searching
Use this procedure to look up calls based on various search criteria.
Procedure 1 Searching for Calls
1.
Go to the Call Lookup page:
Choose VoIP Data > Call Lookup in the Module for VoIP Performance menu.
2.
Click the Search link.
3.
Enter a time period on the General tab (required).
4.
(Optional) Enter additional search criteria on the General tab. See General Search Criteria for details.
5.
(Optional) Click the Advanced tab and enter any Advanced search criteria. See Advanced Search Criteria for details.
6.
(Optional) Change the number of calls to display per page.
7.
(Optional) Exand the “Output format" section to specify the number of calls to display per page (rows) and the metrics to show for individual calls (columns).
8.
Click Go.
A list of calls is shown below the search criteria dialog box. You can print this list or send the information to a text file (in comma-separated value format) by clicking the Print or CSV button.
End of Procedure 1
 
 
Viewing Details About a Single Call or Call Channel
Use this procedure to view details about a specific call or channel:
Procedure 2 Drilling Down to a Specific Call
1.
Enter a Call ID or use Search to display a list of calls containing the call of interest.
A call list is displayed.
2.
Click in a call row to see more details about that call.
Call Info is displayed. You can click the Signaling tab to view a call ladder.
3.
For a list of the channels used by a call, click the “+” beside the call ID.
A list of channels appears.
 
4.
Click in a channel row to see more details about that channel.
Channel information is displayed. To view additional metric information, click on the following tabs:
• 
Call
• 
Quality
• 
Codec
• 
RTP
• 
Burst/Gap
• 
Delay/jitter
• 
Degradation
• 
Misc
• 
Charts
For descriptions of the metrics on these tabs, see Channel Metrics.
End of Procedure 2
 
 
Call List
The call list shows the results of the last call lookup. You can sort the table by clicking on any column heading. Repeated clicks change the sort order between ascending and descending. You can specify which columns appear in the call list from the VoIP Call Metrics Page.
Call Info
Clicking on a call row in the list of calls displays information about that call.

Signaling
To display a call ladder showing the signaling sequence for a call, select a call row in the list of calls and click the Signaling tab.

Statistic Charts
The Statistic Charts link displays charts showing data for the set of currently displayed calls. This link is available when a call list is displayed on the Call Lookup page. The charts shown are
Average number of concurrent calls
Number of started calls per minute
MOS distribution
Degradation factors
