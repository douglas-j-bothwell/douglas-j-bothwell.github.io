NetFlow QoS Insights
This topic describes Insights that use NetFlow data to let you find specific interfaces by application and ToS. These Insights are:
Applications for Selected ToS
Applications for Selected ToS Group
ToS Groups for Selected Application
ToS for Selected Application
Note: To use these Insights, you must have AppResponse release 8.5 or higher and an AppResponse NetFlow license. Also, you must configure ToS data collection, as described in Setting Up NetFlow Data Collection.
