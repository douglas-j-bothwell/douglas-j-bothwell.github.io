User Admin Manager
To access the User Admin Manager, start the Desktop Console, click the Tools menu and select User Admin Manager.
Note: The password for the admin account cannot be changed using the User Admin Manager. The CLI command passwd must be used instead.
Figure 8
User Admin Manager
 
The user admin manager is only available to user accounts with administrative privileges.
To create a new user account, click the New button and supply the name, description, password and privilege level (see also Account Privileges).
Accounts created on a regular appliance are local to that appliance. Whereas, accounts created on the Domain Director are global and can log into any appliance within the domain. However, if a local account exists on an appliance with the same name as a new global account, the appliance rejects the new global account when the account configuration information is distributed by the director. An error appears in the Director Update Log.
