Link User Costs Data Files

Link user costs data files are used to import/export the user-specified cost of a link on a specific layer. The files use the format shown in Figure 5-6. The first row contains four headers; each subsequent row describes a link at a specific layer.

Table 5-4 Link User Costs Data Files
Field
Description
Layer
Layer of the link (OTS, OMS, OCH, or DCL)
From
Name of the source node
To
Name of the destination
Designation (Optional)
The user-defined designation of the link
Cost
User-specified cost of the link
End of Table 5-4
