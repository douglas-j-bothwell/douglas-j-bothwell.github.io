Spreadsheet Reports

In addition to the Bill of Materials, you can also generate reports in .csv (comma-separated-value) files that you can view using a spreadsheet program like Excel. The remaining sections in this chapter describe the different types of reports you can generate and the information contained in each report.

Procedure 15-1 Creating a Spreadsheet Report

  1. Choose Info > Export to Spreadsheet.

    2. The CSV Report Generation dialog box appears.

    Figure 15-6 CSV Report Generation Dialog Box



  2. Select the spreadsheet reports you want to generate and click OK.

End of Procedure 15-1
