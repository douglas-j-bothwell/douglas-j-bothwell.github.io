Grooming DCL Traffic with Optimized Routes

Figure 8-2 shows the Groom DCL to OCH Traffic - Optimized Routes dialog box, which you open by choosing Design > Groom DCL to OCH traffic > Optimized Routes. The Grooming Input group box includes input options for the grooming algorithm: DCL traffic matrix, OCH bit rate, and algorithm. The grooming results appear in the Results and Performance group boxes.

See Workflow Description for instructions for grooming traffic.
