Exporting Browser Information to CSV Files

You can export browser data to a csv (comma-separated vector) file. This makes it easy to view and edit browser data in a spreadsheet application such as Excel. To export browser data, right-click on the parent element in the treeview and choose Export to CSV; this creates a data file that contains all the information in the selected tree element. To export all data in the browser, right-click on the top-level element in the treeview.
