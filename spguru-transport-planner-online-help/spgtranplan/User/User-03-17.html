SONET Nodes: TM

SP Guru Transport Planner supports the use of SDH terminal multiplexers instead of a more expensive DXC to handle SONET traffic. SDH terminal multiplexers aggregate the SONET traffic that terminates in one or more wavelengths within the node. However, it is not possible to switch transit SONET traffic in a TM node. Therefore, the flexibility in a TM node is much lower than in a DXC node (for more information, see Grooming DCL to OCH Traffic ).
