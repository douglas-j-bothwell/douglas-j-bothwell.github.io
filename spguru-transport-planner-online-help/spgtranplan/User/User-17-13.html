Traffic Variations: Batch Options

Table 16-9 lists the options you can specify for a traffic-variations command. For more information, see Traffic Variations.

Table 16-9 Traffic Variations: Batch Options 
Option
Description
Comment
-load
Traffic matrix subject to traffic variation evaluation
Only for OCH traffic matrices
-tvit
Number of iterations
 
-tvpd
% decrease traffic pattern
Value between 0 and 100
-tvpi
% increase traffic pattern
Value between 0 and 100
-tvpir
% increase random
Value between 0 and 100
-tvror
Retain original routing
  • 0 = no
  • 1 = yes
 
End of Table 16-9
