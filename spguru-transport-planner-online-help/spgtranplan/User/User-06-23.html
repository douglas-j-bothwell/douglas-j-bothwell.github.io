Importing and Exporting WDMNetDesign Files

SP Guru Transport Planner can import files from WDMNetDesign v2.7. When you import a file:

  • Imported nodes appear as EOCC nodes. You can change the node type in the OTS Node Browser.

    • For more information, see OTS Node Browser.

  • All nodes have a continuous DXC + OXC. You can change this setting in the Network Properties dialog box.

    • For more information, see Network Properties.

  • SP Guru Transport Planner creates a default WDM line system. The number of wavelengths is equal to the number of wavelengths in the WDMNetDesign file. You can change these settings in the WDM Link Equipment dialog box.

    • For more information, see WDM Link Equipment Properties.

To import a WDMNetDesign project, choose File > Import > WDMND File.
