Editing a Batch File Manually

This section lists the network design operations supported by batch files and the command-line options for each command. Advanced SP Guru Transport Planner users can edit the batch file, using the batch commands and options described in the following sections.

Note—Editing a batch file manually is an advanced feature. You must edit a batch file carefully, to avoid specifying a set of parameters that is not supported by SP Guru Transport Planner.

Actions Supported by Batch Files

Table 16-4 lists the operations that are supported by batch files and includes links to the corresponding section.

Table 16-4 Actions Supported by Batch Files 
Command
Description

Dimensioning: Batch Options

Grooming: Batch Options
