Node Browser

To open the Node Browser, choose Network > Node Browser or double-click on a node in the Project Editor workspace. The information shown in this window depends on the selected network layer, which you can set using the Layer buttons at the top of the browser window. For specific information, see:

  • OTS Node Browser
  • OMS Node Browser
  • OCH Node Browser
  • DCL Node Browser
