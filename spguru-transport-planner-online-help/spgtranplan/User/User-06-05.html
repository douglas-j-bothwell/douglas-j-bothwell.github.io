Node Locations Data Files

Node locations data files use the format shown in Figure 5-4-Node Locations Data File in .csv Format . The first row contains four headers; each subsequent row describes one node.

Table 5-2 Node Locations Data File
Field
Description
Id
Node name
Lat or X
Latitude or X value
Long or Y
Longitude or Y value
Type
Node type, which can be ECC, EOCC, OCC, or CS
End of Table 5-2
