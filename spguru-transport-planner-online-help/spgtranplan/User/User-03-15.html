Node Modeling Capabilities in SP Guru Transport Planner

The type of equipment deployed in a node depends on the node type and some additional properties. This section describes the different node models:

  • SONET Nodes: DXC
  • SONET Nodes: TM
  • SONET Nodes: ADM
  • Optical Nodes: Non-Blocking OXC
  • Optical Nodes: WP-OXC
  • Optical Nodes: OADM
  • SONET/ Optical Nodes: DXC + OXC
  • Integrated SONET/Optical Nodes: IXC
