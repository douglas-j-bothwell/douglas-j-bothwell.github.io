Links View

The By Link view shows the total amount of equipment on each link. The SP Guru Transport Planner types considered on a link are WDMTM, OA, and REG. Table 13-11-Hardware Browser (By Link): Treeview lists the fields that are shown in the links.

Figure 13-10 Hardware Browser (Links View)

Table 13-11 Hardware Browser (By Link): Treeview 
Level
Object
Fields
Comments
1
Link
The browser shows the following information for each link:
  • Name—Link name
  • Devices—Number of devices used by link
  • Cost—Total link cost
To export or import device configuration of a link via a CSV file, right-click on the link and choose Export Devices or Import Devices.
2
Fiber
The browser shows the following information for each fiber:
  • Line System Type—Name of the line system type
  • WDMTMs—Number of WDM terminal multiplexers used by link
  • OAs—Number of optical amplifiers used by link
  • REGs—Number of regenerators used by link
  • Cost—Cost per fiber
If the device configuration is invalid, this level shows the reason why.
3
Device
The browser shows the following information for each device:
  • Name—Device name (Family name/Model name)
  • Vendor—Device vendor
  • Type—Equipment type of device
  • Slots—Number of slots taken
  • Cards—Total number of cards in device
  • Device Cost—Device cost (not including cards)
  • Total Cost—Total device cost (including cards)
  • Design run—Design run during which the device was added
 
4
Card
The browser shows the following information for each card:
  • Name—Card name
  • Cost—Card cost
  • Slots taken—Number of slots taken by card
  • Design run—Design run during which the card was added
 
5
Interfaces/ports
The browser shows the following information for each card:
  • Interfaces—Name of the interface type
  • Ports—Number of ports used/ total number of ports of that interface type on the card
 
End of Table 13-11
