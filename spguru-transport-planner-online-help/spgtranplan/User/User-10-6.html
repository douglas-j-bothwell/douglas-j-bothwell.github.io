Use Existing Capacity/Add New Capacity Settings: Effects

The following settings in the Groom LOP to DCL Traffic Dialog Box affect how the LOP grooming operation uses network capacity:

  • Use existing capacity—If this option is selected, the grooming operation can set up the LOP traffic using existing capacity (non-native DCL connections that are already set up).
  • Add new capacity—If this option is selected, the grooming operation creates a new DCL traffic matrix to support the LOP traffic.
