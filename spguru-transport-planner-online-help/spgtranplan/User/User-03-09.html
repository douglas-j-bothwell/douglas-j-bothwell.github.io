Protection Methods in Meshed Networks

In addition to unprotected routing, SP Guru Transport Planner supports four protection strategies at the OCH layer:

  • 1 + 1 Path Protection
  • Shared Path Protection
  • Link Restoration
  • Path Restoration

The following sections describe the different recovery schemes and the impact on link capacity and node models.

Note—At the DCL layer, SP Guru Transport Planner currently supports 1+1 protection only.
