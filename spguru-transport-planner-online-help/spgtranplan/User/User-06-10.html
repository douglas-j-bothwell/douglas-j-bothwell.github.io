OADM Data Files

OADM data files use the format shown in Figure 5-9. The first row contains eight headers; each subsequent line corresponds to one OADM.

Note—You can import OADMs only if the network is in transparent mode.

Table 5-7 OADM Data File Fields 
Field
Description
Node
The node in which the OADM is located
Node1 / Node2
The end nodes of the links between which the OADM should be added
Index1 / Index2
The indices of the fibers in the links between which the OADM should be added. If the fibers are not equipped yet, they will be equipped automatically as part of the import process.
Type
The Line System type of the OADM
Designation1/ Designation2
If two or more parallel links are present between Node and Node1 (or Node2), Designation1 (or Designation2) indicates which parallel link the specified OADM should use.
End of Table 5-7
