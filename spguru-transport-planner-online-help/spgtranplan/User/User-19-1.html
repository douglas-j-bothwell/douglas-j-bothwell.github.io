Preferences

Preferences provide a uniform system for passing options and parameters. In most cases you can set the values of preferences using Edit > Preferences. The following is a list of preferences specific to SP Guru Transport Planner. Other preferences are not specific to SP Guru Transport Planner. You can search for preferences in the Preferences dialog box.

wdmg_send_to_report_server

Specifies whether SP Guru Transport Planner HTML reports should be sent to the report sever, in addition to being written to the wdmguru.report_location. This preference can be controlled through the SP Guru Transport Planner user interface with the option Info > Send Web Reports to Report Server.

Data Type
boolean
Default Value
FALSE

wdmg_use_SONET

Specifies whether SONET should be used as the default TDM scheme for a new project. If not, SDH will be used as TDM scheme.

Data Type
boolean
Default Value
TRUE
