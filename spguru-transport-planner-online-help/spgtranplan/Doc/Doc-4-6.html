Using the Index

The index contains links to topics, by keywords.

Procedure 3-2 Finding a Topic using the Index

  1. Click the Index tab at the top of the Navigation Frame.
  2. Select the letter at the top to display index entries starting with the selected letter.
  3. Select the index entry of interest.

    4. The corresponding topic appears in the Topic Frame.

End of Procedure 3-2
