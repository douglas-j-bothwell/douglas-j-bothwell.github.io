Topic Frame

The Topic frame displays the topic selected in the Navigation Frame and includes the following buttons:

Button
Name
Description

Show in Contents
Displays the current topic in the Contents.

Previous
Displays the previous topic in help.

Next
Displays the next topic.

Home
Displays the help system home page. The home page includes quick links to the most commonly requested topics. See also Link to help system home page.

PDF
Displays the current topic and related topics in PDF using Adobe Reader. Use this option to print the current topic along with its related topics. (The PDF documentation does not include hyperlinks.)
Note—Adobe Reader must be installed to display topics in PDF.

E-mail
Sends an e-mail to technical support.

Print
Prints the currently displayed topic.

In addition to the buttons, the Topic frame includes the following navigation links:

  • Breadcrumb trail—A breadcrumb trail appears between the Topic frame buttons and the topic text. The breadcrumb trail shows the current topic in relation to the topic hierarchy. Click on a breadcrumb topic link to display the corresponding topic.
  • Link to help system home page—At the end of each topic, there is a link to the first page of the help system.
