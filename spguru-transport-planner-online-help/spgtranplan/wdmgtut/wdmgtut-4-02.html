Link Design

To overcome signal degradation on long fiber links, the signal must be amplified and regenerated at regular intervals. Amplifier and regenerator locations are calculated according to the transmission characteristics of the line system. This is accomplished through the Link Browser dialog box at the OTS layer or the Design > OTS Link Design dialog box. However, in a production network the locations are already known. You can import this information from a data file, as explained in the Importing and Exporting Data chapter of the SP Guru Transport Planner User Guide.

Example import files are located in the following directory: 

<install_dir>\<release>\models\std\wdmguru\examples

Procedure 3-1 Calculating Amplifier and Regenerator Locations

  1. Open the WDMGuru_Examples project.
    1. Select File > Open....
    2. Select the WDMGuru_Examples project, then click Open.

      3. The example project, which contains multiple scenarios, is loaded. The Australia scenario appears in the workspace.

  2. Select Scenarios > Switch To Scenario > USA to load the USA scenario in the workspace.
  3. Do OTS link design to calculate the amplifier and regenerator locations.
    1. Select Design > Link Design....

      2. The Link Design dialog box appears.

      You can select one link or design all the links simultaneously.

    2. Choose ALL LINKS to design all links.
    3. You can design a link for a specific WDM line system type or for ALL LSTYPES (a worst-case link design). For this example, select LH 40-WDM to design for the default line system type (LH 40-WDM). The other options refer to the actual placement of the sites on the link (refer to the SP Guru Transport Planner User Guide).
    4. Select Asymmetrical, Maximum Span Lengths and Mid Span, then press the Design button.



      5. The Link Design Results window appears with a list of sites added on the links.



    5. Close the Link Design Results and the Link Design dialog box.
  4. Inspect the OTS links.
    1. Select Network > Link Browser (or double click on a link) to open the link browser.
    2. Select OTS to inspect the links in the OTS layer. Mark the Show in-line sites option.
    3. Click on the + icon (next to the links) to display the details of the amplifier and regeneration site locations. A yellow icon indicates an amplifier site, while a red icon indicates a regenerator site.
    4. Click on the + icon (next to the sites) to see the functionality performed on the specific fibers in that site.



    5. Close the link browser.
  5. Import the amplifier and regenerator locations.
    1. Select File > Import > Network Elements....
    2. Select OA/Regenerator Locations as the Data Type to import.
    3. Click Import and select the file WDMGuru_import_USA_OA_regen_locations.csv in the directory <install_dir>\<release>\models\std\wdmguru\examples.
    4. After you select the data file, 209 in-line amplifier or regenerator sites are imported. Note that the site locations calculated in the previous paragraph have been overwritten by the imported site locations.



    5. Close the Import Network Elements dialog box.
  6. Inspect the OTS links.
    1. Select Network > Link Browser (or double click on a link) to open the link browser.
    2. Select OTS to inspect the links in the OTS layer. Mark the Show in-line sites option.
    3. Click on the + icon (next to the links) to display the details of the amplifier and regeneration site locations.
    4. Click on the + icon (next to the sites) to see the functionality performed on the specific fibers in that site.
    5. Close the link browser.
  7. Close the project.
    1. Select File > Close. Select Don't Save in the Close Confirm dialog box.

End of Procedure 3-1

Note—To export the OTS in-line sites, use the File > Export > Network Elements... dialog box. Exporting information from a simple network into a file is a good way to discover the required file format in case you want to generate a file by hand, or by some other means.
