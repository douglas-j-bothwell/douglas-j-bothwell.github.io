Licensing

This chapter describes the OPNET analysis software licensing infrastructure, providing an overview of licenses, the licensing system, and the components of the licensing system. OPNET analysis software licensing includes two major components:

  • License Manager is the graphical user interface (GUI) that allows you to interact with License Servers and license files. The License Manager can be used to manage License Servers and license files on local and remote computers.
  • License Server is an application running in the background on a computer.
