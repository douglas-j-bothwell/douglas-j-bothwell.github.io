Solaris and Linux Installation Support

Most sites should follow the installation procedure described on the Solaris or Linux Installation sheet. This procedure uses the installation program to configure your workstation automatically and install SP Guru Transport Planner. However, if your workstation has an unusual or incorrect configuration, the installation program may fail. If so, you can use the information here to solve the problem.
